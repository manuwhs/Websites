Archivos HTML

Aqui podeis colgar archivos de extensio HTML para que colgueis vuestra web, si quereis que aparezcan imagenes u otros archivos
en ellas, primero debeis colgarlos en la web, es decir, los colgais y luego en al archivo HTML poneis la direccion.
El primer examinar, sirve para que pongais el archivo principal de vuestra web para que se muestre,
el segundo sirve para que coloqueis subpaginas y que no se muestren.
Direcciones de los archivos que colgueis:
* Otros archivos HTML ----> "nombre_archivo"
* Fotos ----> "archivo/fotos/nombre_foto"
* Archivos comprimidos ----> "archivo/compri/nombre_archivo"
* Flash ----> "archivo/flash/nombre_animacion"
* Video ----> "archivo/videos/nombre_video"
* Word ----> "archivo/word/nombre_word"
* Musica ----> "archivo/musica/nombre_musica"
