Clases

Las clases en Programación orientada a objetos (POO) son definiciones de los elementos que forman un sistema,
en este caso, definiciones de los objetos que van a intervenir en nuestros programas.
Las clases son declaraciones de objetos, es decir, son un tipo, una clase especifica de objeto, por ejemplo, quieres crear una imagen, pues tienes que crear un ejemplar de la clase imagen, y despues le das las caracteristicas para crearla.
Esto quiere decir que la definición de un objeto es la clase. Cuando programamos un objeto y definimos sus características y funcionalidades en realidad lo que estamos haciendo es programar una clase.
Por lo tanto solo son un conjunto de propiedades y metodos

Una clase es un conjunto de variables, llamados atributos, y funciones, llamadas métodos, que trabajan sobre esas variables. Las clases son, al fin y al cabo, una definición: una especificación de propiedades y funcionalidades de elementos que van a participar en nuestros programas.

Un objeto es un ejemplo de una clase, es decir, una instancia, digamos que la clase esta formada por un conjunto de propiedades y metodos y un objeto es crear un ejemplo de esa clase dandole las propiedades y metodos que queramos.
Cuando se hace una clase simplemente se especifica qué propiedades y funcionalidades tiene.
Por ejemplo, un hombre podría tener como propiedades el nombre o la edad y como funcionalidades, comer, moverse o estudiar.
Las clases, se definien con la funcion "class", que funciona asi ---> class nombre_clase{contenido_clase}.
Entre los corchetes, pondremos las propiedades y funciones.

En la clase hombre declararíamos dos atributos: la edad o el nombre, que serían como dos variables.
Para declarar las variables de una clase, deberemos usar tambien el operador var antes de la variable.
También deberíamos crear tres métodos, con los procedimientos a seguir para que el hombre pueda comer, moverse o estudiar.
Estos métodos se definen declarando funciones dentro de la clase.
El código para definir una clase se puede ver a continuación:

class hombre{
var $nombre;
var $edad;
var $alimentos;
var $sitio;
var $estudios;

function comer($comida){
$this->alimentos = $comida; }

function moverse($destino){
$this->sitio = $destino; }

function mostrar_sitio(){
echo $this->sitio; }

function estudiar($asignatura){
$this->estudios = $asignatura; }
}

En este codigo, hemos creado una clase con varias variables, que estan vacias, y hemos creado cuatro metodos (funciones).
Como veis aparece la variable $this en los metodos, dentro de un método, la variable $this hace referencia al objeto sobre el que invocamos el método.

Instanciar clases

Si queremos trabajar con las clases debemos instanciar objetos, proceso que consiste en generar un ejemplar de una clase.
Por ejemplo, tenemos la clase hombre anterior. Con la clase en si no podemos hacer nada, pero podemos crear objetos hombre a partir de esa clase.
Cada objeto hombre tendrá unas características propias, como la edad o el nombre.
Además podrá desempeñar unas funciones como comer o moverse, ahora bien, cada uno comerá o se moverá por su cuenta cuando le sea solicitado, sin interferir en principio con lo que pueda estar haciendo otro hombre.

Para crear un objeto de la clase, usamos el operador new, la forma para crear un objeto es --->$variable = new nombre_clase
$pepe = new hombre();
$juan = new hombre();

Para usar las propiedades y metodos, usaremos las siguientes estructuras:
nombre_objeto->nombre_propiedad
nombre_objeto->nombre_metodo()
 Ejemplos:
$juan=new hombre;
$juan->alimentos="macarrones";
$juan->moverse("casa");
$juan->mostrar_sitio()

Constructures

Los constructores son funciones, o métodos, que se encargan de realizar las tareas de inicialización de los objetos al ser instanciados. Es decir, cuando se crean los objetos a partir de las clases, se llama a un constructor que se encarga de inicializar los atributos del objeto y realizar cualquier otra tarea de inicialización que sea necesaria. Por ejemplo, sirve para darle un valor inicial a las propiedades del objeto.
