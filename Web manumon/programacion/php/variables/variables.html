Variables

# Una variable es un espacio en memoria donde se almacena un dato, un espacio donde podemos guardar cualquier tipo de información que necesitemos para realizar las acciones de nuestros programas.

# Declarar variables consiste en definir y de paso informar al sistema de que vas a utilizar una variable.
Con PHP, para definir una variable solo necesitamos anteponerle el simbolo del dolar $. Ademas, para usarlas despues tambien tienen que ir precedidas por $ al contrario de lo que pasa con "var" en JavaScript.
Para declarar variables y usar variables usamos la particula $.
Ejemplo:
$numero = 10;
echo $numero

Tipos

El contenido de las variables puede ser de 6 tipos:
Number: puede almacenar números enteros o decimales, positivos o negativos. El signo decimal en PHP es el punto (.). Podrás realizar operaciones matemáticas con estas variables.
Ejemplo: $numero=11
 String: puede almacenar cadenas de caracteres, es decir, un conjunto de caracteres alfanuméricos. Las variables de tipo String deben ir entre comillas dobles (") o simples (').
Ejemplo: $letras = "Solo palabras"
 Boolean: Puede almacenar sólo dos valores, o true (verdadero) o false (falso).
Ejemplo: $verdadero = true / $falso = false

 Cuando no definimos una variable, su valor es undefined. Ejemplo: $nada;

Object: El tipo de datos Object (objeto) se define mediante la clase Object. La clase Object sirve de base para todas las definiciones de clases en ActionScript y le permite organizar unos objetos dentro de otros (objetos anidados).
Ejemplo: $objeto = new Objeto()
 null: El tipo de datos null (nulo) contiene el valor null. Este valor significa "ningún valor", es decir, no hay datos. Puede asignar el valor null en distintas situaciones para indicar que una propiedad o variable no tiene ningún valor asignado.
Ejemplo: $nada = null
 undefined: El tipo de datos undefined (no definido) contiene un valor: undefined. Este es el valor predeterminado de las instancias de la clase Object. Sólo puede asignar el valor undefined a variables que pertenezcan a la clase Object
Ejemplo: $nada = undefined

Arrays: Son variables donde podemos introducir varios valores, en lugar de solamente uno como ocurre con la variables normales.
Los arrays nos permiten guardar varias variables y acceder a ellas de manera independiente, es como tener una variable con distintos compartimentos donde podemos introducir datos distintos.
Para crear un Array antes hay que declararlo:
$datos = new Array()
 Es crea un Array sin contenido alguno, podemos especificar cuantas variables diferentes tendra el Array poniendo su numero entre parentesis. $datos = new Array(2). Esto crea un Array con dos variables de valor undefined ya que no les hemos dado valor.
Las variables en un Array se introducen de forma dinamica de la siguiente manera:
$datos = new Array()
$datos[0] = 155
$datos[1] = "hola"
Como veis la dinamica de las variables de un Array es: nombre_Array [numero_variable] = dato.
Cada variable de un Array es independiente y puede escribirse en ella cualquier dato que se puede escribir en una variable normal.
Tambien podemos escribir entre los parentesis del Array las variables separadas por comas:
$datos = new Array("hola",8,true)
 Aqui tenemos un Array con tres variables, estas variables equivaldrian a:
$datos[0] = "hola"
$datos[1] = 8
$datos[2] = true

Ambito

Se le llama ámbito de las variables al lugar donde estas están disponibles. Por lo general, cuando declaramos una variable hacemos que esté disponible en el lugar donde se ha declarado, esto ocurre en todos los lenguajes de programación y como PHP se define dentro de una página web, las variables que declaremos en la página estarán accesibles dentro de ella. Tipos:

Variables globales: Son las que están declaradas en el ámbito más amplio posible, que en PHP es una página web. Para declarar una variable global a la página simplemente lo haremos en un script, con la palabra var. Ejemplo:
&letras = "Solo palabras"
 Las variables globales son accesibles desde cualquier lugar de la página, es decir, desde el script donde se han declarado y todos los demás scripts de la página.

Variables locales: Son aquellas variables que esten declaradas dentro de una funcion. Solo podemos acceder a ellas en la funcion en la que vallan. Aparte de funciones, son variables locales en general todas aquellas que vallan entre { }. Ejemplo:
function miFuncion (){
$numero = 20;
echo $numero;
}
En este caso numero es una variable local, puede haber una variable local que se llame igual que una global, ya que cada una influye en sus respectivos campos.
En PHP podemos acceder a variables exteriores dentro de una funcion con la particula global:

$numero=20;
function miFuncion (){
global $numero;
echo $numero;
}
