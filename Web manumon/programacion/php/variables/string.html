Cadenas: String

Como ya hemos dicho antes, un String es una variable que almacena cadenas de caracteres, es decir,
un conjunto de caracteres alfanuméricos. Las variables de tipo String deben ir entre comillas dobles (") o simples (').
Esto significa que puede albergar numeros, simbolos y letras.
Ejemplo:
$cadena1 = "Palabras";
$cadena2 = "126832";
$cadena3 = "Tengo 12 cosas ";
Podemos yuxtaponer o concatenar estas variables, es decir, unirlas, para ello usaremos el punto "."
$cadena4 = $cadena1.$cadena1;
echo $cadena4
Esto seria igual a: Palabras 126832
Tambien podemos poner otras variables dentro de nuestro String.
$numero = 20;
$edad = "Tengo $numero años";
echo $edad
Esto es igual a: Tengo 20 años

Ahora vamos a ver caracteres que para ponerlos, debemos escribirlos de otra forma:
CaracterEfecto en la cadena
\$Escribe dolar $ en la cadena
\"Escribe comillas " en la cadena
\\Escribe contrabarra \ en la cadena
\5*2Escribe 5*2 y no 10
CaracterEfecto en la cadena
\tIntroduce una tabulación
\nCrea un salto de linea
\rRetorno de carro
Vamos a terminar diciendo que si sumamos dos Strings que tienen por cadena un numero, se suman.
$numero = "20";
$numero2 = "10";
$suma = $numero + $numero2 echo $suma
Esto pondria en pantalla el numero 30.

$cadena = "Esto es un texto que tiene /n un salto de linea";
