Introduccion

PHP es uno de los lenguajes de lado servidor más extendidos en la web. Nacido en 1994, se trata de un lenguaje de creación relativamente creciente que ha tenido una gran aceptación en la comunidad de webmasters debido sobre todo a la potencia y simplicidad que lo caracterizan.
PHP nos permite embeber su pequeños fragmentos de código dentro de la página HTML y realizar determinadas acciones de una forma fácil y eficaz sin tener que generar programas programados íntegramente en un lenguaje distinto al HTML.
Por otra parte, y es aquí donde reside su mayor interés con respecto a los lenguajes pensados para los CGI, PHP ofrece un sinfín de funciones para la explotación de bases de datos de una manera llana, sin complicaciones.

PHP es un lenguaje de programacion con el que podemos modificar ficheros y nos proporciona un monton de funciones para interactuar con el visitante de forma permanente, es decir que él pueda producir cambios. Entre sus funciones se encuentra:
* Funciones de correo electrónico.
* Gestión de bases de datos
* Gestión de archivos
* Tratamiento de imágenes

Como hemos dicho antes, PHP se puede escribir en un archivo HTML, para ello, el codigo que pongamos debe estar escrito entre las etiquetas:
<?        y      ?>
<%       y      %>
<?php   y      ?>
<script language="php">
Tambien se puede escribir aparte y ponerle la extension .php, para luego cargarlo en el archivo HTML,
para insertarlo despues debemos escribir src="archivo.php" dentro de las etiquetas del lenguaje <script> y </script>.
Asi <script src="archivo.php"></script>

Ahora vamos con la sintaxis, es lenguaje PHP es muy parecido al java, tiene variables y funciones predefinidas.
Para insertar comentarios, los pondremos entre: /* y */ y para poner comentarios solo en una linea, basta con poner #.

PHP es muy estricto en el sentido de la separacion de partes del script, cada vez que creemos una variable, llamemos una funcion... tenemos que terminar con punto y coma ";" en caso contrario, dara error.
Para declarar variables usamos el dolar $ y para crear funciones usamos function(){sentencia}.
En lo referente a la sensibilidad entre mayusculas y minusculas, PHP es sensible a ellas en las variables pero no en las funciones.
