Las Sesiones

En PHP a veces resulta necesario crear variables personales de cada usuario y que se puedan usar variables todo el tiempo tantas veces como queramos. Para eso se crearon las sesiones.
Digamos que las sesiones son una manera de guardar información, específica para cada usuario, durante toda su visita.
Nos sirven para almacenar información que se memorizará durante toda la visita de un usuario a una página web.
Para ello, primeros indicamos que iniciamos sesion y luego declaramos variables.

Sintaxis

Ahora veremos las funciones para manejarse con funciones y como funcionan:
session_start ()

Es funcion inicia la sesion del usuario, puede o crearla o continuarla.
Al hacer session_start() PHP internamente recibe el identificador de sesión almacenado en la cookie o el que se envíe a través de la URL. Si no existe tal identificador se sesión, simplemente lo crea.
Esta funcion debe escribirse antes de la etiqueta html. Ejemplo:
<?
session_start();
?>
<html>
<head>

Una vez iniciada la sesión podemos utilizar variables de sesión a través de $_SESSION, que es un array asociativo.

$_SESSION["variable"]

Esta funcion es la que crea las variables de una sesion, en realidad es un array que almacena las variables de sesion.
Es importante saber que estas variables son diferentes a las demas en que estas se guardan en el navegador. Su estructura es $_SESSION["variable"] = contenido_variable. Ejemplo:
<?
session_start();
?>
<html>
<head>
<title>Generar variable de sesión</title>
</head>
<body>
<?
$_SESSION["mivariabledesesion"] = "Hola este es el valor de la variable de sesión";
?>
</body>
</html>

session_register('variable')

Esta función, además de crear o recuperar la sesión para la página en la que se incluye también sirve para introducir una nueva variable de tipo sesión. Ejemplo:
<?
session_register('contador');
?>
<HTML>
<HEAD>
<TITLE>contador.php</TITLE>
</HEAD>
<BODY>
<?
If (isset($contador)==0)
{$contador=0;}
++$contador;
echo "<a href=\"contador.php\">Has recargado esta página $contador veces</a>";
?>
</BODY>
</HTML>
