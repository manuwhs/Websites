Funciones

Una función es un bloque de código que desempeña una fucnion y que podemos usar con sólo llamarla,
ademas realizan una sentencia y/o devuelven un valor.
Es decir, es un conjunto de variables, operadores y otras funciones. Para empezar una funcion, escribimos 'function', despues el nombre que le des a la funcion, y luego entre () variables indefinidas que usaremos en la funcion y que luego cuando la invoquemos, le demos valor, si no las hay o si no las quieres poner, pones solo ().
Un detalle es que al termiar la funcion, debemos poner antes de } un punto y coma ; asi ;}.
Queda entonces que su expresion general es: function (variables) {sentencia;}.
Una sentencia no es mas que una instruccion, como sumar dos variables, declarar variables o usar otras funciones. Ejemplo:
function miFuncion() {
$miVariable = 1;
echo $miVariable;
}
 Esta funcion define una variable que es igual a 1 y luego la muestra en pantalla.

Para invocar una funcion tenemos que escribir el nombre de la misma seguida de las variables entre parentesis. Ejemplo:
miFuncion()
Al escribir este codigo se ejecutara la funcion, si lo escribimos dos veces, lo hara dos veces, etc.
PHP tambien incorporpora funciones hechas por el llamadas acciones.

Aqui se explica mejor la funcion creadora de funciones "function":

function: Es la funcion creadora de funciones.
Esta funcion sirve para crear tus propias funciones. Funciona asi: function miFuincion(){sentencia;}.
Donde miFuncion es el nombre de la funcion y los parentesis sirven para poner las variables, con las que nos manejaremos en la funcion y que al invocar la funcion, entre los parentesis solo tengamos que poner unos valores que sustituiran a las variables que antes hemos creado. En cado de no querer definir ese tipo de variables solo ponemos ().
Primer ejemplo:
function decirsi () {
echo "si";};
decirsi ()
 Aqui, definimos la funcion "decirsi" cuya sentencia lo que hace es mostrar en pantalla la palabra "si".
En la segunda linea invocamos la funcion.
Segundo ejemplo:
$numero = 10;
$numero2 = 20;
function suma() {
global $numero;
global $numero2;
echo $numero + $numero2;};
suma ()
 Aqui hemos definido dos variables que equivalen a dos numeros, luego hemos creado una funcion que carga esas dos variables e interactua con ellas. En este ejemplo, el resultado seria "30".
Tercer ejemplo:
function suma ($sumando1,$sumando2) {
$sumar = $sumando1+$sumando2 ;
echo $sumar;};
suma(10,3)
Aqui se crea la funcion suma, y dos variables indefinidas entre los parentesis. En la sentencia de la funcion se crea una nueva variable a la que se le da el valor de la suma de las dos variables creadas entre los parentesis y con la sentencia echo se muestra la variable sumar en pantalla. Despues, cuando invocamos la funcion, entre los parentesis introducimos dos valores, que sustituiran a las variables que creamos antes.
El resultado sera mostrar en pantalla la suma de los dos numero que hallamos puesto, en este caso mostrara 13.
