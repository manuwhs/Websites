Base de Datos


Una base de datos es un conjunto de datos estructurados, esta compuesta de tablas donde almacenamos registros catalogados en función de distintos campos (características).
Al igual que PHP, para manejarse con bases de datos necesitaremos un sistema administrador de base de datos, en PHP sirve el Apache y en las bases de datos sirve el MySQL.
Las bases de datos tienen su propio lenguaje estandarizado, el SQL, que forma parte de MySQL.
