Acciones

Las Acciones son funciones predefinidas de PHP, es decir:
PHP las crea, y nosotros sólo tenemos que usarlas de la manera que se nos indica.
No tenemos que definir las funciones ni nada por el estilo, ni siquiera necesitamos saber cómo están hechas... Lo importante es que están listas para usar, lo que facilita el uso de este lenguaje de programación y sobre todo, haga muy rápido comenzar a programar.
Las acciones son practicamento todos los codigos predefinidos por PHP, incluyendo las sentencias, funciones...
Para invocar estas acciones, escribimos su nombre y sus parametros entre () en el caso de que los haya.
Estas acciones, estan contenidas en la libreria de funciones de PHP.

Manipulacion de Strings

strcmp (variable1, variable2)
Esta funcion compara dos variables, es como el operador ==, si las variables son iguales devuelve 0 y si no -1.
<?php
$a = "ola";
$b = "ola";
$comparacion = strcmp($a,$b);
echo $comparacion
?>

 substr(string, inicio, tamaño)
Esta funcion, obtiene una subcadena de un string, el parametro "string" indica el string del que se sustraera la subcadena,
el parametro "inicio" dice desde que caracter se empieza a crear la subacadena y el parametro "tamaño" es opcional
e indica el numero de caracteres que cojeremos.
<?php
$cadena = "Esto es una cadena algo extensa";
$subcadena = substr($cadena,10);
$subcadena2 = substr($cadena,5,12);
echo $subcadena."<br>".$subcadena2
?>

strpos (string, referencia, inicio); strrpos (string, referencia); strstr (string, referencia)
Estas tres funciones nos alludan a saber la situacion de una parte de un string:
- strpos (string, referencia, inicio). Esta funcion devuelve la posicion de una parte de un string, el primer parametro es el string, el segundo el caracter o cadena que buscamos y el tercero es opcional e indica desde donde empezamos a buscar.
- strrpos (string, referencia). Hace lo mismo que la funcion anterior pero empieza buscando de fin a principio.
- strstr (string, referencia). Devuelve la subcadena que haya despues del caracter o cadena que hayamos puesto.
<?php
$cadena= "Esto es una cadena de caracteres";
$posicion1=strpos($cadena,"de");
echo $posicion1."<br>";
$posicion2=strrpos($cadena,"de");
echo $posicion2."<br>";
$subcadena=strstr($cadena,"de");
echo $subcadena
?>

echo cadena y printf (cadena)
Estas dos funciones muestran en pantalla la cadena que especifiquemos.
<?php
$cadena1 = "Cadena mostrada con echo";
$cadena2 = "Cadena mostrada con printf()";
echo $cadena1."<br>";
printf ($cadena2)
?>
