Formularios


Ahora vamos a ver como trabajar con formularios, los formularios son objetos del objeto document asique para referirnos a uno, debemos poner antes document.
Para acceder a un formulario desde el objeto document podemos hacerlo de dos formas:
* A partir de su nombre, asignado con el atributo NAME de HTML.
* Mediante la matriz de formularios del objeto document, con el índice del formulario al que queremos acceder.

Para acceder por ejemplo a este formulario:
<FORM name="form1">
<INPUT type=text name=campo1>
<INPUT type=text name=campo2>
</FORM>
Usaremos esta gerarquia:
document.form1
O esta suponiendo que sea el primer formulario de la web:
document.forms[0]

Si queremos dirigirnos a uno de los campos del formulario usariamos:
document.form1.campo1
o
document.form1.elements[0]


Aqui teneis una serie de objetos de los formularios, aunque antes miraos las propiedades y metodos de form.

Propiedades

action
Es la acción que queremos realizar cuand se submite un formulario. Se coloca generalmente una dirección de correo o la URL a la que le mandaremos los datos. Corresponde con el atributo ACTION del formulario.

elements array
La matriz de elementos contiene cada uno de los campos del formulario.

encoding
El tipo de codificación del formulario

length
El número de campos del formulario.

method
El método por el que mandamos la información. Corresponde con el atributo METHOD del formulario.

name
El nombre del formulario, que corresponde con el atributo NAME del formulario.

target
La ventana o frame en la que está dirigido el formulario. Cuando se submita se actualizará la ventana o frame indicado. Corresponde con el atributo target del formulario.

Metodos

submit()
Para hacer que el formulario se submita, aunque no se haya pulsado el botón de submit.

reset()
Para reinicializar todos los campos del formulario, como si se hubiese pulsado el botón de reset. (Javascript 1.1)
