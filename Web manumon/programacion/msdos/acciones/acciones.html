Etiquetas, Bifurcación y Condiciones


Etiquetas y Bifurcaciones

Una etiqueta es cualquier línea que comienza por dos puntos ":".
Los comentarios en el código batch pueden ser hechos por usando un ":" doble, osea "::"
Las etiquetas son procesadas antes que el resto del codigo.
Las etiquetas que empiezan por un solo ":" sirven para crear puntos de control, es decir,
donde lo pongamos creara una referencia para que con otra funcion podamos saltar hasta esa parte
del codigo para saltarnos otro y que se siga leyendo.
Su esctructura es esta ----> :nombre_punto_control
 La estructura de los comentarios es ----> ::comentario
Ejemplo:
:: esto es un comentario porque empieza por un doble dos puntos, pertenece al grupo de las etiquetas
@echo off
goto punto
*** Este codigo no sera ejecutado***
*** porque nunca sera leido ***
:punto
echo "Hemos saltado ese codigo"
Aqui aparece el comando "goto", este comando crea una bifurcacion, y lo hace llendo al punto de control que pongas despues.
Su estructura es esta ----> goto nombre_punto.
En el ejemplo, creamos el punto ":punto" y una bifurcacion en la tercera linea que hace que el lector de codigo salte donde estaba :punto, por tanto la 4º y 5º lineas no seran leidas.

Condicionales

# Para crear estructuras condicionales en batch, usamos la estructura if, que significa "si".
Su estructura es esta ----> if [condicion] [sentencia]
Recordad que los [ ] no se ponen en realidad, los ponemos en los tutoriales para diferencias las partes.
Lo que quiere decir que si se cumple lo que hallamos puesto en la condicion se ejecutara la sentencia.
Esta estructura admite 6 formas de sintaxis:

if exist fichero orden si existe 'fichero' ejecutar la orden
if not exist fichero ordensi no existe fichero ejecuar la orden
if cadena1==cadena2 orden comparar cadena1 y cadena2
if not cadena1 == cadena2 orden si son distintos
if errorlevel numero orden si el comando anterior acabó con un errorlevel igual o superior a numero ejecutar orden
if not errorleven numero orden si el comando anterior acabó con un errorlevel inferior a numero ejecutar orden
