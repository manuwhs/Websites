Tutoriales SQL


Structured Query Language que no es mas que un lenguaje estándar de comunicación con bases de datos. Como sabemos una base de datos esta compuesta de tablas donde almacenamos registros catalogados en función de distintos campos (características).
Los campos de textos pueden almacenar los tipos de variable normales, como String, numeros, booleanos.

Para crear una tabla debemos especificar diversos datos: El nombre que le queremos asignar, los nombres de los campos y sus características. Además, puede ser necesario especificar cuáles de estos campos van a ser índices y de qué tipo van a serlo.

Sintaxis

Para crear una tabla usaremos la funcion Create que funciona asi.
Create Table nombre_tabla (
nombre_campo_1 tipo_1
nombre_campo_2 tipo_2
nombre_campo_n tipo_n
Key(campo_x,...) )
Para introducir datos en una tabla usaremos la funcion Insert que funciona asi. Insert Into nombre_tabla
(nombre_campo1, nombre_campo2,...)
Values (valor_campo1, valor_campo2...)
Ejemplo:
Insert Into clientes
(nombre, apellidos, direccion, poblacion, codigopostal, email, pedidos)
Values ('Perico', 'Palotes', 'Percebe n°13', 'Lepe', '123456', 'perico@hotmail.com', 33)

Para borrar una tabla o un campo usaremos la funcion Delete.
Para borrar una tabla: Delete From nombre_tabla
 Cuando queremos eliminar solo una parte de la tabla, como por ejemplo lo referente a un campo cuyo nombre es nombre y contenido es "Jesus". Tendriamos que ademas poner la sentencia WHERE seguida del campo y su valor.
Su metodo es asi: Delete From nombre_tabla Where campo = valor
En muchos tutoriales os vendra que despues del Where se pone una condicion, esto tambien es verdad, al poner el campo y el valor eso ya es una condicion ya que si no existe no podra eliminarlo.

Para modificar el contenido de una tabla usaremos la funcion Update que es capaz de cambiar contenidos de los campos. Su estructura es esta: Update nombre_tabla Set nombre_campo1 = valor Where valor_por_el_que_se_cambia.

La selección total o parcial de una tabla se lleva a cabo mediante la instrucción Select. En dicha selección hay que especificar:
-Los campos que queremos seleccionar
-La tabla en la que hacemos la selección
