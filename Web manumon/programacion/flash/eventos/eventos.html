Eventos


Los eventos son aquellas acciones que surgen como respuesta a alguna acción llevada a cabo por el usuario.
Los eventos normalmente estan dirigidos a los clips de pelicula.
Para que un evento este dirigido a un clip de pelicula, debemos poner el nombre del clip mas un "." antes de poner el evento.
Hay muchos eventos diferentes:

onEnterFrame Es un evento lo que hace es repetir la sentencia de una funcion cada fotograma.
Cuando tenemos una accion ya hecha y queremos que la repita cada fotograma, su estructura es esta: onEnterFrame= miFuncion
En caso contrario, podemos definir una sentenci con la siguiente estructura es esta: onEnterFrame=function () {sentencia}.
Ejemplo 1:
var numero=10
onEnterFrame=function () { numero += 1}
 Esto hara que cada fotograma, se le añada uno a la variables numero.
Ejemplo 2:
numero=10
function incrementar(){numero++; trace (numero)}
onEnterFrame = incrementar
 Esto hara que cada fotograma, se le añada uno a la variables numero.
Ejemplo 3:
Clip.onEnterFrame=function () {this._rotation += 5}
Aqui se realiza este evento dentro de un clip de pelicula y hace que cada fotograma gire 5 grados.

onPress: Detecta cuando presionas con el raton sobre un objeto y realiza una sentencia.
En cuanto a la sentencia que realiza, al igual que en el caso anterior, puedes definir una sentencia nueva o introcucir el nombre de la funcion. Su estructura es esta: Clip.onPress = function () {sentencia}. Ejemplo:
function pulsado() {trace("pulsado2")}
Clip.onPress = function () {trace( "pulsado")};
Clip.onPress = pulsado;
Aqui estan los dos ejemplos de como realizar la sentencia, mediante una accion ya definida, y creando una sentencia.
En realidad de las dos funciones, solo se realizaria la ultima osea, que mostraria "pulsado2". Esto es solo los dos tipos de opciones.

onRelease: Detecta cuando sueltas el raton sobre un objeto y realiza una sentencia.
Su estructura es esta: Clip.onRelease = function () {sentencia}. Ejemplo:
function soltado() {trace("soltado2")}
Clip.onRelease = function () {trace( "soltado")};
Clip.onRelease= soltado;
Aqui estan los dos ejemplos de como realizar la sentencia, mediante una accion ya definida, y creando una sentencia.
En realidad de las dos funciones, solo se realizaria la ultima osea, que mostraria "soltado2".

onRollOver: Detecta cuando sueltas el raton se desplaza sobre el clip de pelicula.
Su estructura es esta: Clip.onRollOver = function () {sentencia}. Ejemplo:
function dentro() {trace("dentro2")}
Clip.onRollOver = function () {trace( "dentro")};
Clip.onRollOver = dentro;

onRollOut: Detecta cuando sueltas el raton se desplaza sobre el clip de pelicula.
Su estructura es esta: Clip.onRollOut = function () {sentencia}. Ejemplo:
function fuera() {trace("fuera2")}
Clip.onRollOut = function () {trace( "fuera")};
Clip.onRollOut = fuera;

startDrag / stopDrag: Estas dos acciones, lo que hacen es, la primera arrastrar el objeto con el raton y la segunda parar de arrastrar el objeto con el raton. Ejemplo:
Clip.onPress = function () {this.startDrag()};
Clip.onRelease = function () {this.stopDrag()};
 Este codigo lo que hace es que al pulsar, el raton arrastra el objeto y al soltar ya no.
