Variables



Las variables son contenedores donde podremos almacenar información para trabajar con ella.
Esta información puede ser modificada y leida. Las variables pueden estar precedidas o no de "var".
Este atributo le dara mayor independencia a la variable.

Tipos de variables

Segun el tipo de informacion que contengan existen 8 tipos de variables:
Number: puede almacenar números enteros o decimales, positivos o negativos. El signo decimal en ActionScript es el punto (.). Podrás realizar operaciones matemáticas con estas variables.
Ejemplo: var numero=11
 String: puede almacenar cadenas de caracteres, es decir, un conjunto de caracteres alfanuméricos. Las variables de tipo String deben ir entre comillas dobles (").
Ejemplo: var letras = "Solo palabras"
 Boolean: Puede almacenar sólo dos valores, o true (verdadero) o false (falso).
Ejemplo: var verdadero = true / var falso = false

 MovieClip: Es una variable que almacena un Clip de pelicula para asi poder controlarlo mejor.
Ejemplo: var pelicula = Clip
 Object: El tipo de datos Object (objeto) se define mediante la clase Object. La clase Object sirve de base para todas las definiciones de clases en ActionScript y le permite organizar unos objetos dentro de otros (objetos anidados).
Ejemplo: var objeto = objeto

 null: El tipo de datos null (nulo) contiene el valor null. Este valor significa "ningún valor", es decir, no hay datos. Puede asignar el valor null en distintas situaciones para indicar que una propiedad o variable no tiene ningún valor asignado.
Ejemplo: var nada = null
 undefined: El tipo de datos undefined (no definido) contiene un valor: undefined. Este es el valor predeterminado de las instancias de la clase Object. Sólo puede asignar el valor undefined a variables que pertenezcan a la clase Object
Ejemplo: var nada = undefined
 Void El tipo de datos Void (vacío) sólo contiene un valor: void. Este tipo de datos se usa para designar funciones que no devuelven un valor. Void es un tipo de datos complejo que hace referencia al tipo de datos Void simple.
Ejemplo: var vacio = Void

De todas estas variables, las mas importantes son las tres primeras.
Tambien, podemos indicar de que tipo es una variable cuando la escribimos, aunque no es necesario. Ejemplo:
var numero:Number = 10
var letras:String = "letra"
var sonido:Sound = musica
var Clip:MovieClip = Clip

Ambito de variables

El ambito de las varibles es el sitio donde puedan utilizarse las variables. Existen tres tipos de ambitos:
Linea de tiempo:
Son aquellas que se colocan sobre “la linea de tiempo” ya sea la linea principal, o la de un MovieClip.
Estas variables pueden ser consultadas y modificadas cuantas veces querramos siempre y cuando nos encontremos en dicha linea de tiempo o hagamos referencia a ella.
Ejemplo:
var cinco=5
var saludo=”hola”
 Como veis la primera varibles contiene el numero cinco y la segunda la palabra hola.
Este tipo de variables puede escribirse sin el "var"

Variables locales:
Estas variables solo existen dentro de una función. Ejemplo:

function miFuncion() {
var miVariable= 1;
trace(miVariable);
}
 Podemos observar que agregamos el pre “var” el cual le dara independencia a esta variable. Esta variable con el prefijo “var” se destruira cuando el proceso avandone la función.

Variables globales:
 Estas variables son vistas desde cualquier lugar de nuestro swf y pueden ser consultadas o modificadas siempre.
Para crear una de estas variables, tiene que llevar el prefijo (_global). Ejemplo
_global.numero=8
_global.palabra=”hola”
