# Como os habreis fijado, también está la función "attachBitmap()", que sirve para poner la imagen en el escenario.
Ya hemos visto cómo crear e importar imágenes, estamos listos para modificarlas; este paquete de funciones (import flash.display.*);
también trae algunas para hacerlo como:

1)"draw()":Esta función lo que hace es copiar algo en un mapa de bits, el mapa donde se va a clonar el objeto se pone antes
de "draw()", y el objeto que se clona, se pone en el parentesis así: mapabit.draw(objeto).
2)"clone()":Esta función lo que hace es clonar un mapa de bits. Ejemplo: mapabit2= clone(mapabit1).
Esto significa que mapabit2=mapabit1

3)"threshold()":Esta función sirve para aislar colores de una imagen. Antes del efecto, se pone la imagen a la que se aplica
Y sus parámetros son:
1) La imagen de entrada que se pone, que suele ser una copia de la misma a la que se le aplica.
2) Un rectángulo que define el área al que se le va a aplicar el efecto.
3) Un punto,(ya explicaré lo que es), normalmente de coordenadas (0,0)
4) Un String: "==","!=", "<=",">=","<" o ">"; que varía el efecto sobre los pí
5) Un color sexagesimal con el que se prueba cada píxel para comprobar si está dentro o supera el umbral.
6) Un color en sexagesimal que indica el valor de color al que se define un píxel si se supera la prueba de umbral.
El valor predeterminado es 0x00000000.
7) Un color en sexagesimal que indica la máscara que se va a emplear para aislar un componente de color.
El valor predeterminado es 0xFFFFFFFF.
8) Un valor booleano, si el valor es true, los valores de píxel de la imagen de origen se copian al destino cuando falla la prueba de umbral.
Si es false, la imagen de origen no se copia cuando falla la prueba de umbral. El valor predeterminado es false.

imagen = BitmapData.loadBitmap("imagen");
attachBitmap(imagen, 1)
punto= new Point()
rec=imagen.rectangle
imagen.threshold(imagen, rec, punto, "<", 0xff330000, 0xff808080, 0x00ffffff)
4)"perlinNoise()": Esta función, coge una imagen, o un mapa de bits y crea ruido, es decir, mediante una ecuación cuyos valores son los parametros,
se crean píxeles aleatorios de forma desordenada, da igual el color del mapa de bits o la imagen al que se lo apliquemos,
siempre tendrá el mismo efecto de desconfiguración, ya que no puede tener en cuenta la imagen. Lo que en resumen hace
es deformar un mapa de bits, tu podrás decir cada cuanto se crea un ruido diferente, cuantos ruidos a la vez y que ruido usar;
además el color del resultado sólo podrá ser negro y del color de canal que le pongamos.
Como en el efecto anterior y en muchos otros, antes del efecto hay que poner el mapa de bits al que se aplica,
este tiene nueve parámetros y son:
1) Un numero que indica cada cuantos pixeles el ruido es diferente en el ejede las x.
2) Un numero que indica cada cuantos pixeles el ruido es diferente en el ejede las y.
3) El tercer numero indica el numero de diferentes ruidos que se combinan.
4) La ecución esta del ruido, tiene un valor que es este, que es un termino independiente para que con el mismo tipo de ruido,
se creen diferentes imágenes pero muy parecidas. Normalmente en este parametro se pone un numero aleatorio entre unos
cuantos números, para ser mayor el efecto de aleatoriedad.
5) Es un booleano que si es true intenta suavizar los bordes de transición, y si es false no.
6) Otro booleano que si es true es ruido fractal y si es false genera turbulencia 7)El quinto numero da un canal de color al ruido. Hay 4 canales, hay cuatro canales principales, el 1 (rojo), el 2 (verde),
el 3 (azul) y el 4(alfa). Estos canales de color se pueden mezclar, por ejempor rojo + verde = 1|2
8)Este tercer booleano dice si el color es en blanco y negro
9)El ultimo dato lo que hace es poner el efecto, las funciones individuales de ruido
en un punto, para que si desplazamos ese punto lo que pase es que se valla
generando mas ruido y creando efecto de desplazamiento

import flash.display.*;
mapabit1 = new BitmapData(200,200,false,0x0ff022)
attachBitmap(mapabit1,1)
punto= new Point()
mapabit1.perlinNoise(20,20, 1, 1, false, true, 2, false, [punto])
