# Como ya hemos dicho, este paquete solo sirve para difinir figuras grometricas. Para ello usa:
1) Point(): Esta funcion lo que hace es crear un punto de coordenadas, sirve por ejemplo para colocar objetos o efectos en partes del escenario o la imagen. Entre el parentesis colocamos las coordenadas. Ejemplo:
import flash.geom.*;
punto1=new Point()
punto2=new Point(30,20)
 2) Rectangle(): Esta funcion genera un area, podemos o bien crearla o bien indicamos un objeto del que coger su area.
Para coger el area de un objeto su estructura es esta: var area = mapabit.rectangle.
Si queremos crear uno nuevo, su estructura es esta. var area = new Rectangle (posicion x, posicion y, anchura, altura).
Ejemplo:
import flash.geom.*;
var area = mapabit.rectangle;
var area2 = new Rectangle(0,0,200,300)

2) Matrix(): Esta funcion representa una matriz de transformación que determina cómo asignar puntos de un espacio de coordenadas a otro. Al definir las propiedades del objeto Matrix y aplicarlas a un objeto MovieClip o BitmapData es posible realizar varias transformaciones gráficas en el objeto. Estas funciones de transformación son la traslación (cambio de posición x e y), giro, cambio de escala y sesgo.
Es decir, es una funcion que modifica el tamaño, giro e inclinacion del objeto al que se aplica.
