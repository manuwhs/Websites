DOS

La Unidad dos realiza un gran número de rutinas de utilidad para el tratamiento de archivos.
Estas rutinas permiten ejecutar funciones relacionadas con el sistema operativo.

Rutinas

*Function DiskFree(unidad:word):longint
Proporciona el número de bytes libres en la unidad de disco especificada.
Unidad = 0, indica la unidad por defecto, Unidad = 1, unidad A:, Unidad = 2, unidad B.

*Function DiskSize(unidad:word):longint
Proporciona el número de bytes de capacidad de la unidad. *Procedure Exec(programa, lineaOrdenes:string)
Ejecuta el programa especificado en la línea de órdenes. Realiza una salida temporal hacia el DOS a fin de ejecutar un programa.
Programa = Nombre del programa a ejecutar
lineaOrdenes = Línea de órdenes adicionales al programaEjemplo: Exec(‘\command.com’,’dir *.PAS’);

*Procedure GetDate(var año,mes,dia,diasemana: word)
Devuelve la fecha del reloj interno del sistema operativo

*Procedure GetTime(var hora, minutos, segundos, centésimas:word)
Devuelve la hora actual del sistema operativo (reloj interno)

*Procedure SetDate(año,mes,dia,diasemana : word)
Establece la fecha del reloj interno.

*Procedure SetTime(hora,minutos,segundos,centésima:word)
Establece la hora del sistema operativo.
