String

Los valores que pueden tomar los datos de tipo String son los de secuencias de caracteres alfanumericos
con longitudes que varían desde 1 hasta los 255 caracteres.
Asi que las variables que sean de tipo String solo pueden tener cadenas de carecteres hasta 255.
Estas cadenas van entrecomilladas ''.

Propiedades

Las cadenas solo tienen una propiedad:
* Length
Esta propiedad guarda el número de caracteres del String.

Metodos

* Concat(cadena1,cadena2,cadena3..)
Este metodo concatena las cadenas que haya entre parentesis, es decir las une.

* Pos(cadena1,cadena2)
Sirve para comprovar si la cadena1 esta dentro de la cadena2, es decir, dice si la cadena2 contiene a la cadena1.
Si encuentra la cadena dentro de la otra, devuelve la posicion inicial que ocupa dicha cadena dentro de la otra,
si no lo encuentra devuelve 0.

* Copy(cadena,inicio,cantidad)
Esta funcion devuelve una subcadena de la cadena que pongamos.
La subcadena que devuelve es la subcadena de la cadena principal que empieza por el caracter que pongamos de "inicio"
y a partir de ese caracter, coge tantos caracteres como numero pongamos en "cantidad".

* Insert(cadena1,cadena2, Posicion)
Este metodo introduce la cadena1 en la cadena2 en el lugar que pongamos en Posicion.

* Delete(cadena,inicio,cantidad)
Esta funcion elimina de la cadena dada: Cadena es la cadena de la que se borraran los caracteres,
Inicio es el primer caracter que se borrara y cantidad es el numero de caracteres que se borraran a partir de Inicio.

* Upcase(caracter)
Convierte un caracter en minucula a mayuscula.

* Str(valor,cadena)
El procedimiento Str obtiene una cadena a partir de un valor numérico.
Donde valor es el número a convertir y Cadena es la variable donde se almacenará la cadena ya convertida.
Por ejemplo, si tenemos el numero 269, es procedimiento hara que la cadena valga '269'

* Val(cadena,numero,codigo)
Esta funcion convierte una cadena numerica en un numero.
Donde cadena es la cadena que se convierte, numero la variable numerica a la que sera igualada la cadena hecha numero,
y codigo es una variable de tipo Integer cuyo valor sera 0 si la transformacion ha ocurrido correctamente o su valor
sera la posicion del caracter que haya producido un error.

Trabajar con Strings

Aqui vamos a ver una serie de caracteristicas a la hora de trabajar con Strings:
* Podemos declarar nuevos types de String con la siguiente estructura:
Type
Cadena1 = String[numero]
 Donde numero es la cantidad maxima de caracteres que puede tener una variable de ese tipo.
Asi, si ponemos un numero menos de 255, las variables que usen este tipo ocuparan menos memoria.

* Los caracteres de un String tambien tienen un nombre, como si fueran Arrays, es decir, la estructura de los nombre
de los caractres es esta:
Cadena[numero_caracter]
Donde Cadena es la cadena y numero_caracter la posicion que ocupa el caracter dentro de la cadena.

* Para concatenar varias cadenas de usa el operador +.

Ejemplo

program cadenas;
var
cad1,cad2,cad3,cad4,cad5,cad6:String;
num1,num2,num3,num4,num5:Integer;
car1:char;
begin
cad1 := 'Esta pagina web es una pasada';
cad2 := 'web';
cad3 := '4987';
num1 := 1563;

num2 := Length(cad1);
cad4 := Copy(cad1,6,6);
cad5 := Concat(cad2,cad4);
Val (cad3,num3,num4);
Str (num1,cad6);
num5 := Pos (cad2,cad1);
car1 := Upcase('q');

writeln(cad1); writeln(cad2); writeln(cad3); writeln(num1);
writeln('');
writeln(num2); writeln(cad4); writeln(cad5); writeln(num3);
writeln(num4); writeln(cad6); writeln(num5); writeln(car1);
writeln('');

Delete(cad1,12,num2);
writeln(cad1);
readln;
end.
