Clases (objetos)

Definicion de Clase y Objeto

Las clases son definiciones de los elementos que forman un sistema, en este caso,
definiciones de los objetos que van a intervenir en nuestros programas.
En Pascal las clases son declaraciones de objetos, es decir, son un tipo, una clase especifica de objeto,
por ejemplo, para crear una imagen, tienes que crear un ejemplar de la clase imagen,
y despues le das las caracteristicas para crearla, es decir, decir que color posee cada pixel.

Cuando creamos un objeto y definimos sus características y funcionalidades en realidad lo que estamos haciendo
es programar una clase, es decir, hay una clase y definimos unas caracteristicas que tiene
y ejecutamos sus funcionalidades.

Una clase es un conjunto de variables, llamados atributos, y funciones, llamadas métodos, que trabajan entre si.
Las clases son, al fin y al cabo, una definición: una especificación de propiedades y funcionalidades de elementos
que van a participar en nuestros programas.

Un objeto es un ejemplo de una clase, es decir, una instancia.
Digamos que la clase esta formada por un conjunto de propiedades y metodos
y un objeto es un ejemplo de esa clase dandole las propiedades y metodos que queramos.
Cuando se hace una clase simplemente se especifica qué propiedades y funcionalidades tiene.

Crear clases

Hemos definido y visto lo que son las clases, llegados a este punto os dire que Pascal tiene clases predefinidas,
estas clases son por ejemplo los tipos de datos de las variable (types), cuya instancia (objeto) son las datos.
que entran dentro de su Rango, y las variables por ello, son tambien objetos de las Clases de Datos.
Es decir las variables con un valor de un tipo de dato son objetos de un tipo (clase) de dato cuyo valor
es una propiedad de esa "clase" y tambien por lo tanto tienen metodos que veremos mas adelante.
Las variables, son los objetos de cualquier Clase de Datos

Nosotros tambien podemos crear nuestras propias clases e instanciar objetos de esas clases.
Las clases, se declaran en el apartado de declaracion de tipos de Pascal, la seccion que empieza por type.
En otros capitulos hemos visto como crear tipos de objeto personalizados, pero en realidad creabamos
tipos de datos a partir de otros, es decir, que en realidad eran los mismos, pero ahora crearemos
nuestros propios tipos. La estructura para hacerlo es esta:

TYPE
nombre_clase= CLASS
variable1,variable2....variableN : valor1;
variable3,variable4....variable : valorN;
function nombre_funcion (parametros) : tipo_datos;
function nombre_funcionN (parametros) : tipo_datosN;
procedure nombre_procedimiento (parametros);
procedure nombre_procedimientoN (parametros);
end;
