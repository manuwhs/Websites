Arrays

Un Array es un tipo de variable (type) donde podemos introducir varios valores,
en lugar de solamente uno como ocurre con la variables de tipos normales.
Los arrays nos permiten guardar varias variables y acceder a ellas de manera independiente,
es como tener una variable con distintos compartimentos donde podemos introducir datos distintos.

Array Unidimensional

Un array de una dimensión (unidimensional), también llamado vector o fila, es un tipo de datos estructurado
compuesto de un número determinado de elementos, de tamaño fijo y elementos homogéneos (del mismo tipo).
La característica de tamaño fijo se refiere a que el tamaño del array debe ser conocido en tiempo de compilación.
Es decir al declararlo debemos especificar cuantos subindices tiene el array y de que tipo.
Los subindices son cada uno de los valores que posee un array.

Al declarar que una variable es de tipo Array, a su vez hay que declarar que tipo de Array sera, es decir,
su numero de subindices y su tipo. La estructura de declaracion del tipo Array es esta:
nombre_variable : Array [rango del subíndice] OF tipo;

Donde "rango de subindice" es el numero de subindices expresado en rango que poseera la variable, es decir,
un rango podria ser del 1 al 10 y el Array podria tener 10 valores distintos.
Este "rango de subindice" se indica poniendo el rango entre dos numeros de la siguiente manera X..Y,
donde X es el primer numero y Y el segundo, es decir, por ejemplo para decir el rango del 1 al 10 seria 1..10
Lo que indica el rango son los dos puntos seguidos " .. ".
Y donde "tipo" es el tipo de dato (type) de los subindices del Array. Ejemplo:

Var
Lista : Array [1..50] of integer;

Para nombrar a cada uno de los subindices del Array se pone el nombre de la variable seguido del numero
del subindice entre corchetes: nombre_variable[subindice]
Es decir, en el ejemplo anterior, el subindice 27 por ejemplo se llamaria Lista[27].

Program ejemplo_array;
Var Lista : Array [1..50] of integer;
begin
Lista[1]:=5;
Lista[2]:=10;
Lista[29]:=98;
Lista[45]:=25;
end.
