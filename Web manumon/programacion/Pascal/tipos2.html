Tipos II

El tipo de un dato es el conjunto de valores que puede tomar un dato durante el programa.
Si se le intenta dar un valor al dato fuera del conjunto se producirá un error.
El tipo de una variable es entonces el conjunto de valores que puede tomar durante el programa.

Los tipos de datos son Clases de Datos, y los datos de una Clase de Datos, son ejemplos de esa clase.
Asi pues, las variables son objetos que poseen un dato, este dato ademas es de una Clase de Dato,
asi que las variables son ejemplos de una Clase de Datos.
Cada Clase de Datos tiene un Rango, es decir, un conjunto de valores que pueden tener los datos de ese tipo de datos.
Si un dato no entra dentro de ese Rango entoces no es de esa Clase de Datos

Al declarar un objeto (variables) que tenga un dato (valor) de una Clase de Datos, el ordenador reserva en la memoria principal,
un espacio en bytes igual al tamaño maximo que pueda ocupar un valor de esa Clase de Datos.
Asi pues, cada tipo de dato hara que se ocupe un espacio determinado en memoria.
Por eso nos interesa tambien, crear nuestras propias Clases de Datos ya que por ejemplo,
un dato de la Clase String puede tener como maximo 255 caracteres, lo que se traduce a que se le reservara en memoria,
255 bytes, pero si declarasemos una Clase de tipo String que solo pudiera tener por ejemplo 40 caracteres,
el espacio reservado en memoria seria solo de 40 bytes.
En este segundo apartado de los Tipos de datos, mostraremos como declarar nuevos tipos de datos
y profundizaremos y ampliaremos las Clases de Datos.
