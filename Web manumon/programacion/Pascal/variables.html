Variables

Una variable es un nombre asociado a un elemento de datos que está situado en posiciones contiguas
de la memoria principal, y su valor puede cambiar durante la ejecución de un programa.
Una variable es un contenedor (objeto) donde se almacena un dato, un espacio donde podemos guardar cualquier
clase de información que necesitemos para realizar las acciones de nuestros programas.

Toda variable pertenece a un tipo de dato concreto, en Pascal, cuando declaremos una variable,
se debe indicar el tipo al que pertenece. Así, tendremos variables enteras, reales, booleanas, etc.
Una vez declarada, ya en el cuerpo del programa le daremos un valor, y despues la utilizaremos.

Vida de una variable

Declaracion de variables

Declarar variables consiste en definir y de paso informar al sistema de que vas a utilizar una variable.
Toda variable que vaya a ser utilizada en Pascal tiene que ser previamente declarada.
Para declarar las variables, usaremos la palabra clave var y depues el nombre de las variables y su tipo,
si vamos a declarar mas de una variable de un tipo podemos escribir el nombre de las variables
separadas por comas y despues el tipo.
Para asignarle un tipo a una variable usaremos el operador dos puntos " : ". De esta manera:
var
variable1,variable2,variable3:tipo1;
varianle4, variable5:tipo2;
Ejemplo de declaracion de variables:

var
nombre,apellido,lugar:String;
telefono,direccion:real;

Iniciacion de variables

Esto no es más que darle un valor inicial a una variable.
Así como lo primero que se hace con una variable es declararla, lo siguiente tiene que ser darle un valor.
Esta iniciacion de variables se debe hacer dentro del cuerpo del programa.
Principalmente, existen dos maneras de otorgar valores iniciales a variables:

Mediante una sentencia de asignación

Podemos darle un valor a una variable igualandola a dicho valor mediante el operador " := ". De esta manera:
variable1 := valor1;
variable2 := valor2;
...
Ejemplo:

var
nombre,String;
telefono:real;
begin
nombre := 'Manumonhacker'
telefono := 924251225
end.

Mediante uno de los procedimientos de entrada de datos (read o readln)

Estas dos funciones que daremos mas adelante, lo que hacen es pedir que introduzcamos unos caracteres
mediante el teclado y cuando los hallamos metido, le damos a Enter para continuar.
Su estructura es esta:
read(variable) o readln(variable)
Ejemplo:

var
nombre,String;
telefono:real;
begin
readln(nombre)
readln(telefono)
end.

Utilización de variables

Una vez declarada e iniciada una variable, es el momento de utilizarla.
Esta es la parte que presenta un mayor abanico de posibilidades que ya iremos viendo

Ambito de una variable

Hemos hablado de las variables como contenedores de informacion, ahora vamos hablar de su ambito:
Se le llama ámbito de las variables al lugar donde estas están disponibles.
Por lo general, cuando declaramos una variable hacemos que esté disponible en el lugar donde se ha declarado.
Segun el ambito hay dos tipos de variables:

Variables globales

Son las que están declaradas en el ámbito más amplio posible, es decir,
son las que se pueden utilizar en cualquier parte: en el cuerpo principal y en los subprogramas.
Las variables globales en Pascal son aquellas que estan declaradas fuera de cualquier subprograma, es decir, declaradas donde hemos visto hasta el momento, ya que los subprogramas (rutinas), las daremos mas adelante.
El valor de estas variables puede ser alterado por los procedimientos, pero las variables locales
no pueden ser alteradas por el cuerpo principal ni por otros procedimientos.

Variables locales

Son aquellas variables que esten declaradas dentro de una rutina (subprograma).
Solo podemos acceder a ellas en la rutnia en la que vallan.

Ejemplo de variables locales y globales
PROGRAM Variables;
VAR
Hola : String;
Numero : Integer;

PROCEDURE prueba;
VAR
Adios : String;
BEGIN
Adios := 'Adios, que les vaya bien';
Numero :=20;
WriteLn (Adios); WriteLn (Numero);
END;

BEGIN
Hola := 'Hola, bienvenidos al programa';
WriteLn (Hola);
Numero:=10;
