Acciones

Las Acciones son funciones predefinidas de JavaScript, es decir:
JavaScript las crea, y nosotros sólo tenemos que usarlas de la manera que se nos indica.
No tenemos que definir las funciones ni nada por el estilo, ni siquiera necesitamos saber cómo están hechas... Lo importante es que están listas para usar, lo que facilita el uso de este lenguaje de programación y sobre todo, haga muy rápido comenzar a programar.
Las acciones son practicamento todos los codigos predefinidos por flash, incluyendo los eventos, sentencias, funciones...
Para invocar estas acciones, escribimos su nombre y sus parametros en el caso de que los haya.
Existen distintos tipos de acciones como las sentencias o los eventos, casi todo lo que este predefinido y podamos usarlo son acciones.

Estas acciones, estan contenidas en la libreria de funciones de JavaScript.
Las siguientes funciones son funciones utiles que nos alludaran a manejar datos:

eval(string)
Esta función recibe una cadena de caracteres y la ejecuta como si fuera una sentencia de Javascript.

parseInt(cadena,base)
Recibe una cadena y una base. Devuelve un valor numérico resultante de convertir la cadena en un número en la base indicada.

parseFloat(cadena)
Convierte la cadena en un número y lo devuelve.

escape(carácter)
Devuelve el carácter que recibe por parámetro en una codificación ISO Latin 1.

unescape(carácter)
Hace exatamente lo opuesto a la función escape.

isNaN(número)
Devuelve un boleano dependiendo de lo que recibe por parámetro. Si no es un número devuelve un true, si es un numero devuelve false.

Ejemplos

eval(string)
Como ya hemos explicado antes, esta funcion ejecuta una cadena de caracteres como si fuera una sentencia de JavaScript.
Esto nos brinda la posibilidad de escribir una variable que sea una cadena de texto y pos lo tento no haga nada y que luego la podamos ejecutar con esta funcion. Esta funcion ejecutaria por ejemplo una suma que estuviera en String. Ejemplo:
<script>
var yo="3+2";
eval("document.write(" + yo +")")
</script>
 Analicemos este codigo: Creamos la variable yo que es un String, osea texto plano, y despues usamos la funcion eval,
dentro de los parentesis tiene que haber un String osea que tiene que empezar por comillas o ser una variables.
Como sabemos, cuando abrimos unas comillas, las siguientes que pongamos seran de cierre, sabiendo esto, vemos que
las primeras comillas contienen "document.write(" y las otras ")", lo unico que queda fuera es + yo +, la variables de String yo,
Tiene que aparecer fuera de las comillas o sino, el navegador interpretara que es texto plano, por eso aparece fuera de las comillas
y los signos + lo que hacen es unir las partes.
Asique el codigo equivaldria a:
document.write(3 + 2)

parseInt(cadena,base)
Esta variable tiene dos parametros, la primera es un numero, que escribiremos entre comillas dobles y la segunda es la base que utilizaremos para saber el valor de ese numero en base 10 osea en forma decimal.
Esta funcion permite formas de base 2 (binario), 8 (octal), 10 (decimal), 16 (hexadecimal). Ejemplo:
<script>

parseInt("100",16)
"Devuelve el numero 256"

parseInt("100",10)
"Devuelve el numero 100"

parseInt("100",8)
"Devuelve el numero 64"

parseInt("100",2)
"Devuelve el numero 4"

</script>
 En caso de que no fuera un numero lo que introducimos, nos dara como resultado sNaN.
Si quieres ver los resultados, usa write() asi: document.write (parseInt("100",numero))

parseFloat(cadena)
Esta funcion lo unico que hace es convertir una cadena en un numero.
<script>
var numero="25";
document.write(parseFloat(numero))
</script>

escape(carácter)
Como hemos dicho antes esta funcion convierte el tipo de un texto en codificación ISO Latin .
<script>
document.write(escape("<Hola>"))
</script>
 Este codigo escribiria: %3CHola%3E

unescape(carácter)
Hace lo contrario que escape.
<script>
document.write(unescape("%3E"))
</script>
 Este codigo escribiria: >

isNaN(número)
Esta funcion devuelve true si lo que hay entre parentesis no es un numero y false si lo es:
<script>
var numero = isNaN("12")
var numer = isNaN("FFFFFF")
document.write(numero +" / "+ numer)
</script>
 Esto daria como resultado false / true
