Variables

# Una variable es un espacio en memoria donde se almacena un dato, un espacio donde podemos guardar cualquier tipo de información que necesitemos para realizar las acciones de nuestros programas.

# Declarar variables consiste en definir y de paso informar al sistema de que vas a utilizar una variable.
Es una costumbre habitual en los lenguajes de programación el definir las variables que se van a usar en los programas y para ello, se siguen unas reglas estrictas. Pero javascript se salta muchas reglas por ser un lenguaje un tanto libre a la hora de programar y uno de los casos en los que otorga un poco de libertad es a la hora de declarar las variables, ya que no estamos obligados a hacerlo, al contrario de lo que pasa en la mayoría de los lenguajes de programación. De todos modos, es aconsejable declarar las variables, además de una buena costumbre y para ello Javascript cuenta con la palabra var. Como es lógico, se utiliza esa palabra para declarar la variable antes de utilizarla.
Para declarar variables usamos "var".
Ejemplo:
var numero = 10

Tipos

El contenido de las variables puede ser de tres tipos:
Number: puede almacenar números enteros o decimales, positivos o negativos. El signo decimal en ActionScript es el punto (.). Podrás realizar operaciones matemáticas con estas variables.
Ejemplo: var numero=11
 String: puede almacenar cadenas de caracteres, es decir, un conjunto de caracteres alfanuméricos. Las variables de tipo String deben ir entre comillas dobles (").
Ejemplo: var letras = "Solo palabras"
 Boolean: Puede almacenar sólo dos valores, o true (verdadero) o false (falso).
Ejemplo: var verdadero = true / var falso = false

 Cuando no definimos una variable, su valor es undefined. Ejemplo: var nada;

Arrays: Son variables donde podemos introducir varios valores, en lugar de solamente uno como ocurre con la variables normales.
Los arrays nos permiten guardar varias variables y acceder a ellas de manera independiente, es como tener una variable con distintos compartimentos donde podemos introducir datos distintos.
Para crear un Array antes hay que declararlo:
var datos = new Array()
 Es crea un Array sin contenido alguno, podemos especificar cuantas variables diferentes tendra el Array poniendo su numero entre parentesis. var datos = new Array(2). Esto crea un Array con dos variables de valor undefined ya que no les hemos dado valor.
Las variables en un Array se introducen de forma dinamica de la siguiente manera:
var datos = new Array()
datos[0] = 155
datos[1] = "hola"
Como veis la dinamica de las variables de un Array es: nombre_Array [numero_variable] = dato.
Cada variable de un Array es independiente y puede escribirse en ella cualquier dato que se puede escribir en una variable normal.
Tambien podemos escribir entre los parentesis del Array las variables separadas por comas:
var datos = new Array("hola",8,true)
 Aqui tenemos un Array con tres variables, estas variables equivaldrian a:
datos[0] = "hola"
datos[1] = 8
datos[2] = true

Ambito

Se le llama ámbito de las variables al lugar donde estas están disponibles. Por lo general, cuando declaramos una variable hacemos que esté disponible en el lugar donde se ha declarado, esto ocurre en todos los lenguajes de programación y como javascript se define dentro de una página web, las variables que declaremos en la página estarán accesibles dentro de ella. Tipos:

Variables globales: Son las que están declaradas en el ámbito más amplio posible, que en Javascript es una página web. Para declarar una variable global a la página simplemente lo haremos en un script, con la palabra var. Ejemplo:
var letras = "Solo palabras"
 Las variables globales son accesibles desde cualquier lugar de la página, es decir, desde el script donde se han declarado y todos los demás scripts de la página.

Variables locales: Son aquellas variables que esten declaradas dentro de una funcion. Solo podemos acceder a ellas en la funcion en la que vallan. Aparte de funciones, son variables locales en general todas aquellas que vallan entre { }. Ejemplo:
function miFuncion (){
var numero = 20
alert (numero)
}
En este caso numero es una variable local, puede haber una variable local que se llame igual que una global, ya que cada una influye en sus respectivos campos.
