Funciones

Una función es un bloque de código que podemos utilizar en cualquier parte del archivo con sólo llamarla, ademas realizan una sentencia y/o devuelven un valor.
Para empezar una funcion, escribimos el codigo function, despues el nombre que le des a la funcion, y luego entre () las variables externas que se puedan poner en la funcion, si no las hay o si no las quieres poner, pones solo (). Ejemplo:
function miFuncion() {
var miVariable = 1;
alert (miVariable);
}
 Esta funcion define una funcion que es igual a 1 y luego la muestra el numero 1.

Para invocar una funcion tenemos que escribir el nombre de la misma seguida de las variables entre parentesis. Ejemplo:
miFuncion()
Al escribir este codigo se ejecutara la funcion, si lo escribimos dos veces, lo hara dos veces, etc.
JavaScript tambien incorporpora funciones hechas por el llamadas acciones.

Aqui se explica mejor la funcion creadora de funciones "function":

function: Es la sentencia de sentencias.
Esta sentencia sirve para crear tus propias funciones. Funciona asi: function miFuincion(){sentencia}.
Donde miFuncion es el nombre de la funcion y los parentesis sirven para crear nuevas variables, con las que nos manejaremos en la funcion y que al invocar la funcion, entre los parentesis solo tengamos que poner unos valores que sustituiran a las variables que antes hemos creado. En cado de no querer definir ese tipo de variables solo ponemos ().
Primer ejemplo:
function decirsi () {alert("si")}
decirsi ()
 Aqui, definimos la funcion "decirsi" cuya sentencia lo que hace es mostrar la palabra "si".
En la segunda linea invocamos la funcion.
Segundo ejemplo:
numero = 10
numero2 = 20
function suma() {trace(numero+numero2)}
suma ()
 Aqui hemos definido dos funciones que equivalen a dos numeros, luego hemos creado una funcion que interactua con es esas dos variables, entoces carga esas dos variables en la funcion y la ejecuta. En este ejemplo, el resultado seria "30".
Tercer ejemplo:
function suma (sumando1,sumando2) { var sumar = sumando1+sumando2 ; return sumar}
alert(suma(10,3))
Aqui se crea la funcion suma, y dos variables indefinidas entre los parentesis. En la sentencia de la funcion se crea una nueva variable a la que se le da el valor de la suma de las dos variables creadas entre los parentesis y con la sentencia return se devulve el valor de la variable suma. Despues, cuando invocamos la funcion, entre los parentesis introducimos dos valores, que sustituiran a las variables que creamos antes. Ademas lo que hacemos es mostrar con la accion "alert" la funcion. Mostrar 13.
