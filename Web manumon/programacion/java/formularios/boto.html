Botones de Radio


El botón de radio es un elemento de formulario que permite seleccionar una opción y sólo una, sobre un conjunto de posibilidades.
Se consiguen con la etiqueta <INPUT type=radio>.
Aunque para que solo podamos elegir uno de los botones, todos tienen que tener el mismo nombre.

Propiedades

checked
Indica si está chekeado o no un botón de radio.

defaultChecked
Su estado por defecto.

value
El valor del campo de radio, asignado por la propiedad value del radio.

Length (como propiedad del array de radios)
El número de botones de radio que forman parte en el grupo. Accesible en el vector de radios.

Metodos

blur()
Retira el foco de la aplicación del campo de texto.

focus()
Pone el foco de la aplicación en el campo de texto.

click()
Es como si hiciésemos un click sobre el checkbox, es decir, cambia el estado del checkbox.

Ejemplos


<INPUT type=radio name="deportes" value="Futbol" onClick=alert(this.value)> Futbol
<INPUT type=radio name="deportes" value="Baloncesto" onClick=alert(this.value)> Baloncesto
<INPUT type=radio name="deportes" value="Tenis" onClick=alert(this.value)> Tenis
<INPUT type=radio name="deportes" value="Boleybol" onClick=alert(this.value)> Boleybol
