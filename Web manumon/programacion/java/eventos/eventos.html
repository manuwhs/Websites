Eventos


Los eventos son aquellas acciones que surgen como respuesta a alguna acción llevada a cabo por el usuario.
Para referirnos a un evento en HTML, el nombre del evento irá precedido por "on".
Por ejemplo, el gestor de eventos de "Click" será "onClick".

Normalmente, estos eventos se colocan dentro de las etiquetas HTML de algun objeto, esto hace que el evento se refiera e ellos, por ejemplo si lo colocas dentro de las etiquetas de una imagen y es el evento onClick pues entonces cuando cliquees sobre la imagen se generara la sentencia.
Su estructura es esta onEvento = "sentencia". Como ves, la sentencia se escribe entre comillas dobles, la sentencia puede escribirse ahi o puedes invocar una funcion ya definida.

Los eventos estan destinados a una serie de objetos, un evento no se puede utilizar en todos los objetos, por ejemplo el evento onKeyPress solo puede darse en campos de texto.

EventoSituacion que detecta.
Click Cuando se pulsa sobre el objeto indicado
DblClickCuando se pulsa dos veces sobre el objeto indicado
AbortCuando se interrumpe un objeto que se esta cargando
DragDropCuando se arrastrar un objeto a la ventana del navegador
ErrorCuando se produce un error en la carga de un documento
FocusCuando el usuario se posiciona en una ventana o cuadro de texto de un formulario
KeyDownCuando se pulsa una tecla
KeyPressCuando se pulsa o libera una tecla
KeyUpCuando se libera una tecla
LoadCuando se carga un documento en el navegador
MouseDownCuando se pulsa un botón del ratón en el objeto indicado
MouseMoveCuando se mueve el cursor en el objeto indicado
MouseOverCuando el puntero del ratón se posiciona en el objeto indicado
MouseOutCuando el puntero del ratón sale de un enlace o imagen mapa
MouseUpSe libera un botón del ratón sobre el objeto indicado
MoveCuando se mueve la ventana. Esta acción también la puede realizar el script
ResetCuando se pulsa sobre el botón reset del formulario
ResizeCuando las dimensiones de la ventana cambian
SelectCuando se selecciona una de las opciones de un cuadro combo del formulario
SubmitCuando se pulsa el botón submit del formulario
UnLoadCuando el usuario sale de la página

Ejemplos

1- Aqui creamos una funcion y luego hacemos que se realice al pulsar sobre el objeto que es la imagen:
<script>
<!--
function Muestramensaje() {
alert("Esto es un mensaje, activado por una función javascript"); }
//-->
</script>

 <img src="imagen.jpg" onClick="Muestramensaje ()">

2- Aqui usaremos el evento onLoad en la etiqueta body para que al cargarse muestre un mensaje.
<script>
<!--
function Muestramensaje() {
alert("Esto es un mensaje, activado por una función javascript"); }
//-->
</script>

 <body onLoad="Muestramensaje ()">

 3- Aqui usaremos onUnload para con el body de este frame para que nos muestre un mensaje cuando nos vallamos o lo cerremos.
