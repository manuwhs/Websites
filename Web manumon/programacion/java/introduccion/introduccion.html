Introduccion

JavaScript

JavaScript es un lenguaje de programacion interpretado por el navegador sin necesidad de un servidor.
JavaScript es un lenguaje de programacion de tipo script (alto nivel)
por lo que dispone de variables, métodos y objetos predefinidos.
Es un lenguaje cuyo paradigma de progrmacion es basicamente basado en Objetos y orientado a Eventos.
Es un lenguaje basado en objetos y esta hecho para ser interpretado por lo que no generan ningun archivo ejecutable.

Javascript es un lenguaje de programación utilizado para crear pequeños programitas
encargados de realizar acciones dentro del ámbito de una página web.
Con Javascript podemos crear efectos especiales en las páginas y definir interactividades con el usuario.
El navegador del cliente es el encargado de interpretar las instrucciones Javascript
y ejecutarlas para realizar estos efectos e interactividades,
de modo que el mayor recurso, y tal vez el único, con que cuenta este lenguaje es el propio navegador.

¿Donde se escribe el codigo JavaScript?

El codigo JavaScript se puede escribir dentro de una pagina .HTML o aparte en un archivo .js :
* Para insertar codigo JavaScript en un documento .HTML, es necesario insertar las etiquetas <script> y </script>
dentro de la cabecera o dentro del cuerpo de la página, es decir, entre <head> y <head> o entre <body> y </body>
Entre estas dos etiquetas <script> y </script> escribiremos el codigo JavaScript en la pagina

* Tambien, podemos utilizar el codigo JavaScritp de un archivo .js que hallamos creado,
asi, no tenemos que escribir el codigo en el arhivo html.
Para importar el archivo se introduce el atributo src="archivo.js" puesto dentro de las etiquetas <script> y </script>.
 Asi <script src="archivo.js"></script>

Sintaxis

* JavaScript es sensible al contexto, por lo que difrencia entre mayusculas y minusculas.
* Es un lenguaje basado en objetos (no orientado a objetos) lo que significa que existe una jerarquia objetos:
- Que tienen propiedades y metodos
- Que sufren eventos
- Y que pueden definir que accion realizar cuando se produce el evento.
* Como cualquier lenguaje de programacion de alto nivel, JavaScript cuenta con variables, que son espacios de memoria
donde se almacena un dato, las variables no han de tipificarse, es decir, no hace falta declarar el tipo de dato que almacenaran
y cualquier variable puede cambiar su tipo de dato si lo igualamos a otro.
* En JavaScript no se pueden declarar constantes.
* Las instrucciones de nuestros Scripts se han de separar mediante ' ; ' o con un salto de linea
