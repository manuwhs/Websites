Crear Clases


Ahora vamos a aprender a crear nuestras propias clases para asi poder crear nuestros propios objetos.
Para crear una clase debemos crear una funcion especial, que se encargará de construir el objeto en memoria e inicializarlo.
Esta función se le llama constructor en la terminología de la programación orientada a objetos.
El nombre de estas funcion tiene que empezar por mayuscula y su estructura es esta:
function MiClase (valor_inicializacion) {
//Inicializo las propiedades y métodos
this.miPropiedad = valor_inicializacion
this.miMetodo = nombre_de_una_funcion_definida }

Utiliza la palabra this para declarar las propiedades y métodos del objeto que se está construyendo. This hace referencia al objeto que se está construyendo, pues recordemos que a esta función la llamaremos para construir un objeto. A ese objeto que se está construyendo le vamos asignando valores en sus propiedades y también le vamos asignando nombres de funciones definidas para sus métodos. Al construir un objeto técnicamente es lo mismo declarar una propiedad o un método, solo difiere en que a una propiedad le asignamos un valor y a un método le asignamos una función.

Trabajar con propiedades

Ya que sabemos como crear propiedades, debemos saber que les podemos dar un valor inicial o que dicho valor lo indique el creador del objeto al crearlo. Estas propiedades pueden ser modificadas y explotadas.

Vamos a hacer un ejemplo de propiedades:
function AlumnoUniversitario(nombre, edad){
this.nombre = nombre
this.edad = edad
this.universidad = null }
Aqui hemos creado el objeto AlumnoUniversitario, al que le podemos poner dos datos de entrada, y hemos creado tres propiedades. Ahora crearemos un objeto de esta clase:
var alumno = new AlumnoUniversitario ("Manuel", 20)
alumno.universidad = Badajoz
document.write (alumno.nombre)
 Ahora creamos un objeto de esta clase, le ponemos los parametros que se pueden poner y luego mostramos en pantalla la propiedad nombre del objeto alumno.

Trabajar con metodos

Para crear un metodo, antes debemos crear una funcionm que se contruye con el objetivo de usarla de metodo en esa clase.
Una vez creada la funcion, el metodo se indica el la clase igual que la funcion: this.metodo = nombre_funcion

Vamos a ver un ejemplo sobre los metodos:
<script>
function matriculate (num_matricula) {
this.numMatricula = num_matricula }

function imprimete(){
document.write("
Nombre: " + this.nombre)
document.write("
Edad: " + this.edad)
document.write("
Número de matrícula: " + this.numMatricula) }

function AlumnoUniversitario(nombre, edad){
this.nombre = nombre
this.edad = edad
this.numMatricula = null
this.matriculate = matriculate
this.imprimete = imprimete}
</script>
 Vamos a ver, el primer parrafo es una funcion que tiene un paramtro de entrada, esta funcion hace que el parametro de entrada sea igual la propiedad numMatricula de la clase en la que valla como metodo.
El se segundo parrafo lo que hace es mostrar en pantalla propiedades de la clase en la que valla como metodo.
El tercero es la clase, en ella hay dos paramtros de entrada que se convierten en el resultado de dos propiedades, despues hay otra propiedad sin definir y por ultimo hay dos metodos cuyas funciones estan ya definidas.

Ahora crearemos un objeto de esta clase:
<script>
alumno = new AlumnoUniversitario("Hector", 22)
alumno.matriculate(521)
alumno.imprimete()
</script>
Hemos creado un objetode la clase AlumnoUniversitario y con los dos parametros iniciales hemos definido dos propiedades. Despues hemos invocado un metodo de la clase y le hemos dado el paramtro 521, este metodo lo que hara darle a la propiedad this.numMatricula el valor del numero que hallamos puesto entre los parentesis.
Por ultimo hemos usado el metodo imprimete() para mostrar en pantalla los datos del alumno.

Como habreis visto, a la hora de delcarar la funcion, ponemos la funcion sin los parentesis "()" y a la hora de usarlo si.
