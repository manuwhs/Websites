Tablas


Ahora aprenderemos a trabajar con las tablas, estas, son muy utiles en cuanto a diseño,
porque todos los objetos se alinean por defecto a la izquierda de las páginas web,
pero gracias a las tablas es posible distribuir el texto en columnas, colocar imágenes al lado de un bloque de texto,
y otra serie de cosas que sin las tablas serían imposibles de realizarse.

Las tablas de invocan mediante las etiquetas <table> y </table>.
Entre dichas etiquetas habrá que especificar las filas y columnas que formarán la tabla.
Para insertar una fila, escribimos las etiquetas <tr> y <tr>
 Pero no basta con especificar las filas, sino las columnas, entre cada etiqueta <tr> y <tr>, tendremos que escrbir tantas veces
<td> y </td> como columnas queramos que tenga dicha fila. Es necesario poner las etiquetas <td> y </td> incluso cuando la fila solo esta formado por una columna. Y entre cada par de etiquetas <td> y </td> escribimos el texto que queramos
Ejemplo de tabla:
<table>
<tr><td>Casa</td><td>Perro</td></tr>
<tr><td>Coche</td><td>Agua</td></tr>
<tr><td>Fuego</td><td>Tierra</td></tr>
</table>
CasaPerro
CocheAgua
FuegoTierra

Atributos

Como veis el resultado no es muy estetico, por eso <table> tiene atributos para modificar el aspecto de la tabla:

Atributo
Significado
Posibles valores

width
ancho de la tabla

un número, acompañado de % cuando se desee que sea en porcentaje
height
altura de la tabla
un número, acompañado de % cuando se desee que sea en porcentaje
cellpadding
espacio entre el contenido de las celdas y el borde
un número
cellspacing
espacio entre celdas
un número
border
grosor del borde
un número
align
alineación de la tabla dentro de la página
left (izquierda)
right (derecha)
center (centro)
bgcolor
color de fondo
número hexadecimal
background
imagen de fondo
número hexadecimal
bordercolor
color del borde
número hexadecimal
Ejemplo:
<table width = 600 height = 400 bordercolor = #ffffff bgcolor = #00ff00 cellpadding = 10 cellspacing = 0>
<tr><td>Casa</td><td>Perro</td></tr>
<tr><td>Coche</td><td>Agua</td></tr>
<tr><td>Fuego</td><td>Tierra</td></tr>
</table>

Atributos de las celdas

Tembien es posible, modificar las caracteristicas de las columnas con los atributos:

Atributo
Significado
Posibles valores

width
ancho de la tabla

un número, acompañado de % cuando se desee que sea en porcentaje
height
altura de la tabla
un número, acompañado de % cuando se desee que sea en porcentaje
align
alineación horizontal del contenido de la celda
left (izquierda)
right (derecha)
center (centro)
valign
alineación vertical del contenido de la celda
baseline (línea de base)
bottom (inferior)
middle (medio)
top (superior)
bgcolor
color de fondo
número hexadecimal
background
imagen de fondo
número hexadecimal
bordercolor
color del borde
número hexadecimal

Ejemplo:
<table>
<tr><td width="50%" border="2" align="center" cellspacing="0" bordercolor="#000000" bgcolor="#FFCC99">Casa</td><td>Perro</td></tr>
<tr><td>Coche</td><td>Agua</td></tr>
<tr><td>Fuego</td><td>Tierra</td></tr>
</table>

Para la etiqueta <td> tambien existen dos atributos, colspan y rowspan. A través del atributo colspan se especifica el número de columnas por las que se extenderá la celda, y a través del atributo rowspan se especifica el número de filas por las que se extenderá la celda.
