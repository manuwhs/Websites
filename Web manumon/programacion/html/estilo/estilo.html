Hojas de Estilos


Como os habreis fijado, no podemos cambiar muchos aspectos de nuestra web, para ello se crearon los CSS u hojas de estilo.
El modo de funcionamiento de las CSS consiste en definir, mediante una sintaxis especial, la forma de presentación que le aplicaremos a:
Un web entero, de modo que se puede definir la forma de todo el web de una sola vez.
Un documento HTML o página, se puede definir la forma, en un pequeño trozo de código en la cabecera, a toda la página.
Una porción del documento, aplicando estilos visibles en un trozo de la página.
Una etiqueta en concreto, llegando incluso a poder definir varios estilos diferentes para una sola etiqueta.
Esto es muy importante ya que ofrece potencia en nuestra programación. Podemos definir, por ejemplo, varios tipos de párrafos: en rojo, en azul, con margenes, sin ellos...

Ademas con los CSS podremos definir un monton de cosas mas que en HTML.
Podemos definir la distancia entre líneas del documento.
Se puede aplicar identado a las primeras líneas del párrafo.
Podemos colocar elementos en la página con mayor precisión, y sin lugar a errores.
Y mucho más, como definir la visibilidad de los elementos, margenes, subrayados, tachados...

# Para definir y aplicar estilos, podemos hacer dos cosas:
Crear una clase y luego insertarla con el atributo class="clase"
Definir el estilo directamente donde lo queramos aplicar con el atributo style=""
Definir el estilo de una etiqueta o elemento de HTML .

Sintaxis

# Vamos a ver, pera definir un estilo se utilizan atributos como font-size,text-decoration... segudos de dos puntos y el valor que le deseemos asignar. Podemos definir un estilo a base de definir muchos atributos separados por punto y coma. Ejemplo:
font-size: 10pt; text-decoration: underline; color: black

# A la hora de definir una etiqueta como por ejemplo <h1> o crear una clase de estilo, tenemos que escrbirlo dentro de las etiquetas
<style> y </style>, pero para definirlo dentro de un sitio concreto, como por ejemplo una fila de una tabla solo necesitaremos poner dentro de la etiqueta el atributo style="" y dentro de las comillas escribir los atributos de CSS. Ejemplo:
<style>
h1 {color:ff00ff;font-size:20; text-decoration: underline; text-align:center}
</style>

<select name="menu" style="Background-color:000000;color:cornsilk; font-size:17">
<option> Uno </option>
<option> Dos </option>
<option> Tres </option>
<option> Cuatro </option>
<select>

En el primer parrafo, hemos definido la etiqueta h1, cada vez que la invoquemos tendra esas propiedades.
En el segundo parrafo, hemos definido el estilo de un menu, le hemos puesto otro color y tamaño a la letra y fondo.

# Ahora vamos a crear una clase, una clase es un conjunto de atributos de style="", es como una funcion en Javascript, sirve para no tener que volver escribir el estilo cuando lo apliquemos a diferentes partes.
Su sintaxis de creacion es: .nombre_clase{atributos}
Como veis es casi igual a definir una etiqueta solo que con un "." al principio, las clases tambien se escriben entre <style>.
Una vez definido la clase, lo podemos a aplicar a un texto, objeto... con el atributo class = "nombre_clase".
<style>
.letrafondo {Background-color:000000;color:cornsilk; font-size:17}
</style>

<select name="menu" class="letrafondo">
<option> Uno </option>
<option> Dos </option>
<option> Tres </option>
<option> Cuatro </option>
<select>

Aqui hemos definido la clase letrafonfo y luego la hemos usado en el menu.
La ventaja que tiene esto es que puedes aplicarla en el objeto que quieras y si no, no; mientras que si defines la etiqueta, todo lo que hagas con esa etiqueta tendra esas caracteristicas.

Atributos

Bueno, ya hemos visto su sintaxis, ahora veremos sus diferentes atributos para cambiar el estilo de elementos.

Atributo Descripcion Valores
color Sirve para indicar el color del texto RGB o nombre de color
font-size Sirve para indicar el tamaño del texto Unidades CSS
font-family Sirve para indicar la tipografia del texto. Fuentes
font-style Es el estilo de la fuente. normal, italic o oblique
font-weight Indica la anchura de la fuente, es decir, lo "negrilla" que es normal, bold, bolder, lighter o 100 - 900

Atributo Descripcion Valores
line-height Indica el alto de una linea y por tanto, el espaciado entre líneas. normal o unidades CSS
text-decoration Para establecer la decoración de un texto underline, overline o line-through
text-align Sirve para indicar la alineación del texto. left, right, center o justify
text-indent Sirve para hacer sangrado o márgenes en las páginas. Unidades CSS
text-transform Sus valores indican si poner el texto en mayus o minus capitalize, uppercase, lowercase, none

Atributo Descripcion Valores
Background-color Indica el color de fondo Color
Background-image Imagen de fondo Direccion de la imagen

Atributo Descripcion Valores
Margin-left Indica el tamaño del margen izquierdo. Unidades CSS
Margin-right Indica el tamaño del margen derecho Unidades CSS
Margin-top Indica el tamaño del margen de arriba. Unidades CSS
Margin-bottom Indica el tamaño del margen de abajo. Unidades CSS
Padding-left Espacio izquierdo entre el borde del objeto y su interior Unidades CSS
Padding-right Espacio derecho entre el borde del objeto y su interior Unidades CSS
Padding-top Espacio de arriba entre el borde del objeto y su interior Unidades CSS
Padding-bottom Espacio de abajo entre el borde del objeto y su interior Unidades CSS
Border-color Color del borde Unidades CSS
border-width Anchura del borde. Unidades CSS
float Alinea un texto a la derecha o izquierda de un objeto none | left | right
clear Hace lo contrario que el anterior none | left | right one
