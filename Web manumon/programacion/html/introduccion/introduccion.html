Introduccion

HTML es un lenguaje informatico de marcas e interpretado por el navegador.
Es un lenguaje de hipertexto, es decir, un lenguaje que permite escribir texto de forma estructurada,
esta compuesto por etiquetas, que marcan el inicio y el fin de cada elemento del documento.
Un documento hipertexto no sólo se compone de texto, puede contener imágenes, sonido, vídeos, etc...
que podemos introducir con las etiquetas correspondientes
El HTML (Hyper Text Markup Language) es el lenguaje con el que se escriben las páginas web.

Los documentos HTML deben tener la extensión html o htm, para que puedan ser interpretados y visualizados
por los navegadores (programas que permiten visualizar las páginas web).
Los navegadores se encargan de interpretar el código HTML de los documentos,
y de mostrar a los usuarios las páginas web resultantes del código interpretado.

HTML no es un lenguaje de programacion, es el lenguaje que sirve como escenario para introducir en él texto,
y los archivos de imagen, audio, video...

Editores de HTML

Un editor es un programa que nos permite redactar documentos,
con ellos escribimos los lenguajes informaticos como el HTML, ya que estos lenguajes funcionan con codigo y el codigo es texto.
Existen un gran número de editores que permiten crear páginas web sin la necesidad de escribir ni una sola línea de código HTML. Este tutorial sirve para no tener que usarlos, usaremos pues uno de los editores más sencillos que hay, el block de notas
Para que sea interpretado el codigo HTML, el formato del archivo que contiene el codigo debe ser .html o .htm.
Si no podeis cambiar el formato de un archivo, podreis poner operativa esta opcion realizando los siguientes pasos:
* Abrid una carpeta
* Darle en el menu de arriba a Herramientas/Opciones de carpeta
* Le dais al apartado de Ver y por las ultimas opciones encontrareis una que diga: Ocultar extension de archivos. La quitais y listo.

Para crear un nuevo documento de texto, osea se un archivo de Block de Notas clickeamos sobre el escritorio
o una carpeta con el boton derecho y le damos a "Nuevo ---> Documento de text".
El formato de este documento es .txt asi que cuando termines de escribir el codigo deberas cambiarlo a .HTML.
Una vez sabido donde escribir el codigo y como hacer para que el navegador lo interprete estamos listo para aprender este lenguaje.

Lenguaje HTML <Etiquetas>

En un documento HTML existen las llamadas etiquetas que son la parte de el codigo que realiza instrucciones.
Estas etiquetas se encuentran siempre entre los signos "menos que y mas que" osea "< y >".
Entre estos dos simbolos se pone el identidicador de la etiqueta, que es su nombre, por ejemplo el identicador de <html> es html.
Generalmente, por cada identificador (etiqueta) que realiza una tarea, existen dos etiquetas, la de comienzo y la de fin o cierre:
* La etiqueta de comienzo está delimitada por los caracteres < y >.
Está compuesta por el identificador o nombre de la etiqueta, y puede contener una serie de atributos opcionales
que permiten añadir ciertas propiedades. Su sintaxis es: <identificador atributos>
* La etiqueta de final está delimitada por los caracteres < /y >.
Está compuesta por el identificador o nombre de la etiqueta, y no contiene atributos. Su sintaxis es: </identificador>
Hay exepciones de esta regla que ya veremos mas adelante

Podriamos clasificar las etiquetas en dos tipos, segun la tarea que desempeñen:
* Las que modifican el texto y los elementos (su color, disposicion, tamaño), como <font> o >center<.
* Las que declaran elementos como la cabecera, el titulo o el cuerpo... (<head>, <body>)
o añaden elementos como tablas, listas, imagenes, videos, archivos de sonido.

Estructura de una pagina HTML

La estructura básica donde se escribe el codigo HTML es esta:
 <html>
<head>
...
</head>
<body>
...
</body>
</html>

# Ahora veamos cómo funcionan estas etiquetas:
<html> y </html>:
Todas las paginas web tienen que tener las etiquetas <html> y </html>.
Entre estas etiquetas tiene que estar comprendido el codigo. Estas etiquetas le dicen al navegador como interpretar el codigo.
<head> y </head>:
Estas etiquetas sirven para guardar informacion en ellas e introducen la cabecera.
Entre estas etiquetas estan comprendidas otras como <link>, <style> o <title>. Lo que escribamos en entre esta última etiqueta (<title> ..... </title>),sera lo que aparecerá en la parte superior de la ventana del navegador, cuando la página esté cargada.
<body> y </body>:
El cuerpo del documento contiene la información propia del documento, es decir, contiene todos los elementos que queramos que se visualizen tienen que encontrarse entre estas etiquetas, que van justo debajo de la cabecera.
Ademas, estas etiquetas tienen algunos que indican caractristicas generales de la pagina:
 <body bgcolor="#0000FF"> Indica el color de fondo de la pagina, tiene que ser un numero hexadecimal
<body background="fondo.gif"> Indica la imagen de fondo de la pagina, si la imagen no ocupa toda pagina, de multiplica
<body text="#FF0000"> Con esta etiqueta se fijan las propiedades del texto normal, como por ejemplo el color.
<body body leftmargin="20" topmargin="20"> Entre los bordes de la pagina y el texto existe un margen, estos atributos lo definen.
