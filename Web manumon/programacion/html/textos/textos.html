Textos y Etiquetas

# Ya sabemos la estructura de una pagina, en este tema aprenderemos las dintintas etiquetas que sirven para modificar nuestro textos.
Primero decir que la fuente predefinida de texto es negra, de tamaño 3 y Times New Roman.
Los textos no se muestran tal cual los escribimos, por ejemplo, para crear un salto de liea se necesita poner una etiqueta.
Aunque una etiqueta <pre> y </pre>, hace que el modo en el que escribamos el texto entre estas etiquetas sera igual al modo en el que salga en pantalla, aunque esto provoca no poder usar otras etiquetas entre estas.

Etiqueta <font>

* Para configurar las propiedades más elementales del texto usamos la etiqueta <font>. Esta etiqueta tiene los siguientes atributos:

Atributo
Significado
Posibles valores

face
fuente

nombre de la fuente, o fuentes
color
color del texto
número hexadecimal o texto predefinido
size
tamaño del texto
valores del 1 al 7, siendo por defecto el 3,
o desplazamiento respecto al tamaño por defecto,
añadiendo + o - delante del valor

Ejemplo:
<font color="#993366" size="4" face="Comic Sans MS, Arial, MS Sans Serif">Texto de color verde</font>

* Existen una serie de etiquetas que permiten aplicar diferentes estilos al texto que se encuentra entre ellas, y generalmente se utilizan para resaltarlo. Estos estilos pueden agruparse según vayan asociados al tipo de letra o a la información que represente el texto. No hay que olvidar que todas estas etiquetas necesitan una etiqueta de cierre, y que pueden aplicarse varias etiquetas al mismo texto.

Herramientas

* Ahora vamos a ver una serie de estiquetas que nos alludan a moldear el texto:
<br>No necesita etiqueta de cierre y crea un salto de linea
<hr>No necesita etiqueta de cierre y crea una barra de la cual pues cambiar los atributos
<blockquote> y </blockquote>Crea un sangrado de texto, es decir un gran espacio

* Ahora vamos a la etiqueta <p> y </p> que separan el texto en parrafos. Cuando se cierra esta etiqueta, se crea un salto de linea automatico. Esta etiqueta tiene atributos, que sirven para alinear el texto que lleva dentro, estos atributos tambien su utilizan en otros aspectos de HTML que ya veremos más adelante. Este atributo es align y es igual a tres valores:

- <p align="center">: Centra el parrafo
- <p align="left">: Pone el parrafo en la parte izquierda
- <p align="right">: Pone el parrafo en la parte derecha

Comentarios

Por ultimo veremos como poner comentarios en el arhivo, estos comentarios no se ven a la hora de ejecutarlo y sirven para que cuando revisemos el codigo sepamos masomenos que parte es cual e indicarnos algunas cosas.
Asi pues, para insertar un comentario hay que ponerlo entre <!-- y //-->.
