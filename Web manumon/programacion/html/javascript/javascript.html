JavaScript



JavaScript es un lenguaje de programacion de tipo script por lo que dispone de variables, métodos y objetos predefinidos.
Es un lenguaje basado en objetos y esta hecho para ser interpretado por lo que no generan ningun archivo ejecutable.
Para insertar funciones JavaScript en un documento, es necesario insertar las etiquetas <script> y </script> dentro de la cabecera o dentro del cuerpo de la página Es decir, entre <head> y <head> o entre <body> y </body>.

Supongamos que un navegador no es capaz de leer la etiqueta <script>, entoces lo que haria seria mostrar un pantalla el codigo que hemos escrito. Para que esto no ocurra, podemos insertar el codigo entre las etiquetas de los comentarios <!-- y //--> para que si no lee, no muestre nada.

Un ejemplo practico seria este:
<script>
<!--
function Muestramensaje() {
alert("Esto es un mensaje, activado por una función javascript"); }
//-->
</script>

 <img src="imagen.jpg" onClick="Muestramensaje ()">

En este codigo, he creado una funcion llamada Muestramensaje que tiene la funcion alert que muestra el mensaje que queramos en forma de alerta. Luego hemos insertado una imagen y le hemos puesto el evento onClick, este evento lo que hace es detectar la accion de presionar sobre algo, y como esta puesto en la imagen, cuando presionemos sobre la imagen, pasara lo que pongamos a continuacion de ese evento. Y dice que al presionar se realice la funcion Muestramensaje(), por lo que al presionar sobre la imagen, se abrira el mensaje de alerta.
