Vinculos e Hipervinculos

Un hiperenlace, hipervínculo, o vínculo, no es más que un enlace, que al ser pulsado lleva de una página o archivo.
Aquellos elementos (texto, imágenes, etc.) sobre los que se desee insertar un enlace han de encontrarse entre las etiquetas <a> y </a>.
A través del atributo href se especifica la página o archivo a la que está asociado el enlace, la página que se visualizará cuando el usuario haga clic en el enlace.

Por ejemplo, para insertar el enlace:
Visita esta pagina web
Habría que escribir:
<a href="http://manumonhacker.iespana.es/">Visita esta pagina web</a>

Este tipo de referencia se llama referencia absoluta y te lleva a un documento externo, es decir una ubicacion de internet.
Como es habreis fijada hay que escribir la direccion entre comillas dobles e incluyendo http://
El otro tipo de referencia es la referancia relativa y sirve para abrir documentos dentro propios, que esten junto a este en la misma carpeta o en una subcarpeta. Me explico, una web esta formado por un conjunto de archivos y carpetas, relacionados entre sí, todos ellos se encuentran en la carpeta principal llamada carpeta raiz. Para poder acceder a un archivo, debe estar como minimo dentro de la misma carpeta que este archivo .html.

Por ejemplo, para insertar el enlace:
Imagen que esta en la misma carpeta
Habría que escribir:
<a href="imagen.jpg">Visita esta pagina web</a>
En caso de que el archivo de destino, estubiese en una subcarpeta, entoces escribiriamos:
<a href="carpeta/imagen.jpg">Visita esta pagina web</a>

Destinos

El destino del enlace determina en qué ventana va a ser abierta la página vinculada, se especifica a través del atributo target al que se le puede asignar los siguientes valores:

_blank:Abre el documento vinculado en una ventana nueva del navegador.
_parent: Abre el documento vinculado en la ventana del marco que contiene el vínculo o en el conjunto de marcos padre.
_self: Es la opción predeterminada. Abre el documento vinculado en el mismo marco o ventana que el vínculo
_top: Abre el documento vinculado en la ventana completa del navegador.

Ejemplos:
<a href="http://manumonhacker.iespana.es/" target="_blank">Visita esta pagina web</a>
<a href="http://manumonhacker.iespana.es/" target="_parent">Visita esta pagina web</a>
<a href="http://manumonhacker.iespana.es/" target="_self">Visita esta pagina web</a>
<a href="http://manumonhacker.iespana.es/" target="_top">Visita esta pagina web</a>

Color de los vinculos

Los colores de los vínculos pueden especificarse a través de las propiedades de la página, en la etiqueta <body>.
Estos colores se asignan a través de los atributos link (vínculo), alink (vínculo activo), y vlink (vínculo visitado).
* link permite determinar el color de los enlaces sin visitar (enlace que no ha sido pulsado ninguna vez).
* alink permite determinar el color del enlace activo (enlace que acaba de ser pulsado).
* vlink permite determinar el color de los enlaces visitados (enlaces que ya han sido pulsados).
Ejemplo: <body link="#FF0000" vlink="#FF0099" alink="#FF9900">

Anclas

Cuando se tienen documentos extensos, divididos en varios apartados, es preferible poder ir directamente al apartado
deseado como ir al comienzo del documento.
Para ello se utilizan las anclas, o puntos de fijación, muy útiles a la hora de crear índices.
Un ancla necesita que se inserten las <a > y </a >, con el atributo name, que puede tomar cualquier valor inventado por el usuario, se recomienda no utilizar caracteres especiales.
Para insertar un punto de fijacion escribimos por ejemplo:
<a name="miancla"></a>
Para referirnos a este punto de fijacion, y dirigirnos a esa parte concreta de la pagina usamos el siguiente codigo:
<a href="#miancla">Enlace al ancla</a>
Si clickeamos sobre el texto "Enlace al ancla" iremos a parar a la parte donde pusimos el ancla. Ejemplo:
Ir arriba
