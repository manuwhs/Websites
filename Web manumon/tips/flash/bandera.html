Este efecto combina los efectos perlinNoise
y DisplazamentMapfilter para lograr hacer
este efecto bandera. Las explicaciones las
encontrareis en el archivo FLA y el
codigo que se usa está a derecha,
cambiando unos parámetro podremos hacer
efecto casacada o lago en vertical.



En este archivo flash encontrareis algunas
explicaciones, este efecto crea distorsion
y convulsusiones; y las aplica a una
imagen, y poniendo esos efectos en
puntos, el efecto se desplaza.
si mirais los tutoriales de esta pagina
lo entendereis bien.
Pero si quereis tenerlo sin mas,
teneis que importar una imagen al
archivo .fla y vincularla a AS2
con el nombre: imagen.
Pero deciros que aquí se dibuja
una imagen dentro de otra para
que se mueva toda la imagen de dentro. 		import flash.display.BitmapData;
import flash.geom.*;
import flash.filters.DisplacementMapFilter

imagen = BitmapData.loadBitmap("bandera");

ancho = imagen.width;
alto = 4*imagen.height/3;

mapaimagen = new BitmapData(ancho, alto);
mapabit = new BitmapData(ancho, alto);

mapaimagen.draw(imagen, new Matrix(1, 0, 0, 1, 0, alto/7));

createEmptyMovieClip("pelicula", this.getNextHighestDepth());
pelicula.attachBitmap(mapaimagen, 0);
pelicula._y=100
pelicula._x=100

punto = new Point();
desMap = new DisplacementMapFilter(mapabit, null, 8, 8, 24, 20, "clamp");

onEnterFrame = function () {

mapabit.perlinNoise(ancho/2, 2*alto, 1, 0, true, true, 8, true, [punto]);

punto.x -= 20
punto.y -= 10

pelicula.filters = [desMap];}
