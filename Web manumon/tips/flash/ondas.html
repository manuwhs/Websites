Este efecto combina los efectos perlinNoise
y DisplazamentMapfilter para lograr hacer
este efecto onda. Las explicaciones las
encontrareis en el archivoFLA y el
codigo que se usa está a derecha,
cambiando unos parámetro podremos hacer
efecto casacada o lago en vertical.



En este archivo flash encontrareis algunas
explicaciones, este efecto crea distorsion
y convulsusiones; y las aplica a una
imagen, y poniendo esos efectos en
puntos, el efecto se desplaza.
si mirais los tutoriales de esta pagina
lo entendereis bien.
Pero si quereis tenerlo sin mas,
teneis que importar una imagen al
archivo .fla y vincularla a AS2
con el nombre: imagen 		import flash.display.*;
import flash.geom.*;
import flash.filters.*;

imagen = BitmapData.loadBitmap("imagen");

ancho = imagen.width;
largo = imagen.height;

createEmptyMovieClip("pelicula", 0)
pelicula.attachBitmap(imagen, 1);
punto = new Point();
punto2= new Point();
punto3 = new Point();
mapabit1 = new BitmapData(ancho, largo/6);
mapabit2 = new BitmapData(ancho, largo/3);
mapabit3 = new BitmapData(ancho, largo/2);
desp1 = new DisplacementMapFilter(mapabit1, new Point(0, 0), null, 4, 0, 20, "clamp");
desp2 = new DisplacementMapFilter(mapabit2, new Point(0, largo/6), null, 4, 0, 50, "clamp");
desp3 = new DisplacementMapFilter(mapabit3, new Point(0, largo/2), null, 4, 0, 120, "clamp")

onEnterFrame = function () {

mapabit1.perlinNoise(0, 2, 1, 0, true, true, 4, true, [punto]);
mapabit2.perlinNoise(40, 4, 2, 0, true, true, 4, true, [punto2]);
mapabit3.perlinNoise(80, 8, 1, 0, true, true, 4, true, [punto3]);

punto.y -= 0.2;
punto2.y -= 0.4;
punto3.y -= 0.8;
pelicula.filters = [desp1,desp2,desp3]}
